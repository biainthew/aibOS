### 조건부서식


**Q) 제 01060 호 에서 세 번째 숫자가 0이다**


⇒ 숫자 중 세번째가 아니라 그냥 세번째 자리에 있는 숫자


{% raw %}
```sql
MID(문자, 3, 1)="0"
```
{% endraw %}


⇒ mid(문자,3번째부터,1개) = “0” (mid 는 숫자도 문자열로 인식)


### 배열수식


**Q) 환자 구분을 이용하여 환자분류별 진료건수만큼 별 표현**


{% raw %}
```sql
=REPT("★",COUNT(IF($G$3:$G$24=$A28,1)))
```
{% endraw %}


⇒ G3 = A28, G3 = A29, G3 = A30, …
     G3 = A29, …
     이런식으로 들어가야 하므로 배열수식! 그리고 카운트 할 수 있게 true 일 때 1 넣어줌


**Q) 조제월별, 환자 구분 별 약제비 총액 합계**


{% raw %}
```sql
=SUM(IF((MONTH($A$3:$A$24)=$F28)*($G$3:$G$24=G$27),$J$3:$J$24))
```
{% endraw %}


⇒ 조건 둘 다 만족해야 하므로 * 넣어주고 true 일 때 값은 약제비 총액


> 💡 **& 와 * 차이**  
> & 는 문자열로 연결  
> * 는 와일드카드, 조건 두 개 일 때 연결하는 용도
