1. 로컬 사용자 및 그룹 선택
2. 사용자 폴더 선택 후 마우스 우클릭 -> 새 사용자
    1. ID = 사용자 이름 : ICQA
    2. 전체이름 : 전체관리자
    3. 설명 : 한국정보통신자격협회
    4. Password = 새 암호, 암호확인 : ICQAPass
    5. 암호 변경할 수 없음 : 다음 로그온 시 사용자가 반드시 암호를 변경해야 합니다 체크박스 해제, 사용자가 암호를 변경할 수 없음 체크박스 선택
    6. 암호 사용 기간 제한 없음 : 암호 사용 기간 제한 없음 체크박스 설정
    -> 만들기
3. 사용자 폴더에 만든 ICQA 계정 선택 후 마우스 우클릭 -> 속성
    1. 소속그룹 -> 추가 -> 지금 찾기 -> Administrators 선택 -> 확인
    2. 로컬경로 = 프로필의 홈 폴더의 로컬 경로에 작성 : C:\icqa 입력 -> 확인
