## 1. 문자열 메서드


### 공백 제거 메서드


**`strip()`** : 문자열 양쪽 끝의 공백을 모두 제거


{% raw %}
```python
text = "   PYTHON   "
result = text.strip()
print(result)       # "PYTHON"
print(len(result))  # 6
```
{% endraw %}


**`lstrip()`** : 문자열 왼쪽 끝의 공백만 제거


{% raw %}
```python
text = "   PYTHON   "
result = text.lstrip()
print(result)       # "PYTHON   "
print(len(result))  # 9
```
{% endraw %}


**`rstrip()`** : 문자열 오른쪽 끝의 공백만 제거


{% raw %}
```python
text = "   PYTHON   "
result = text.rstrip()
print(result)       # "   PYTHON"
print(len(result))  # 9
```
{% endraw %}


---


### 문자열 연결/분리 메서드


**`join()`** : 리스트의 요소들을 특정 구분자로 연결하여 하나의 문자열로 침


{% raw %}
```python
data = ["2025", "01", "15"]
result = "-".join(data)
print(result)           # "2025-01-15"
print(result.count("-")) # 2
```
{% endraw %}


**`split()`** : 문자열을 지정한 구분자를 기준으로 분리하여 리스트로 반환


{% raw %}
```python
text = "HELLO-WORLD-PYTHON"
parts = text.split("-")
print(parts)        # ["HELLO", "WORLD", "PYTHON"]
print(parts[1])     # "WORLD"
print(len(parts))   # 3
```
{% endraw %}


---


### 문자열 변경 메서드


**`replace()`** : 문자열 내의 특정 문자열을 다른 문자열로 치환


{% raw %}
```python
text = "BANANA"
result = text.replace("A", "O")
print(result)           # "BONONO"
print(result.count("O")) # 3
```
{% endraw %}


**`upper()`** : 문자열의 모든 문자를 대문자로 변환


{% raw %}
```python
text = "python"
result = text.upper()
print(result)  # "PYTHON"
```
{% endraw %}


**`lower()`** : 문자열의 모든 문자를 소문자로 변환


{% raw %}
```python
text = "PYTHON"
result = text.lower()
print(result)  # "python"
```
{% endraw %}


**`swapcase()`** : 문자열의 대소문자를 서로 변환


{% raw %}
```python
text = "PyThOn"
result = text.swapcase()
print(result)         # "pYtHoN"
print(result.upper()) # "PYTHON"
```
{% endraw %}


---


### 문자열 포맷팅


**`format()`** : 문자열 내의 중괄호 `{}`에 값을 삽입


{% raw %}
```python
name = "PYTHON"
version = 3
result = "Language: {}, Version: {}".format(name, version)
print(result)  # "Language: PYTHON, Version: 3"
```
{% endraw %}


**`% 포맷팅`****:** % 연산자를 사용하여 값을 삽입


{% raw %}
```python
str1 = "R AND %s" % "STR"
print(str1)  # "R AND STR"

str2 = "REMEMBER NOVEMBER"
result = str2[:3] + str2[12:16]
str3 = "R AND %s" % "STR"
print(result + str3)  # "REMEMBEDR AND STR"
```
{% endraw %}

## 2. 리스트 메서드


### 추가/삽입 메서드


**`append()`** : 리스트 끝에 요소 하나를 추가


{% raw %}
```python
data = [1, 2]
data.append(3)
print(data)  # [1, 2, 3]
```
{% endraw %}


**`extend()`** : 리스트 끝에 다른 리스트의 모든 요소를 추가


{% raw %}
```python
data = [1, 2]
data.extend([3, 4])
print(data)  # [1, 2, 3, 4]
```
{% endraw %}


**`insert()`** : 특정 위치에 요소를 삽입


{% raw %}
```python
data = [1, 2, 3, 5]
data.insert(3, 4)  # 인덱스 3에 4 삽입
data.insert(0, 0)  # 인덱스 0에 0 삽입
print(data)  # [0, 1, 2, 3, 4, 5]
```
{% endraw %}


---


### 제거 메서드


**`pop()`** : 특정 인덱스의 요소를 삭제하고 반환, 인덱스를 생략하면 마지막 요소를 제거


{% raw %}
```python
data = [1, 2, 3]
value = data.pop(1)  # 인덱스 1 제거
print(value)  # 2
print(data)   # [1, 3]
```
{% endraw %}


**`remove()`** : 특정 값의 첫 번째 요소를 삭제


{% raw %}
```python
data = [1, 2, 3, 2]
data.remove(2)  # 첫 번째 2만 제거
print(data)  # [1, 3, 2]
```
{% endraw %}


---


### 정렬/순서 메서드


**`reverse()`** : 리스트 요소의 순서를 뒤집어 반환


{% raw %}
```python
data = [1, 2, 3]
data.reverse()
print(data)  # [3, 2, 1]
```
{% endraw %}


**`sort()`** : 리스트를 오름차순으로 정렬


{% raw %}
```python
data = [3, 1, 2]
data.sort()
print(data)  # [1, 2, 3]
```
{% endraw %}


---


### 검색 메서드


**`count()`** : 특정 값의 개수를 반환


{% raw %}
```python
data = [1, 2, 3, 2, 4, 2, 5]
count_result = data.count(2)
print(count_result)  # 3
```
{% endraw %}


**`index()`** : 특정 값의 첫 번째 인덱스를 반환


{% raw %}
```python
data = [1, 2, 3, 2, 4, 2, 5]
index_result = data.index(2)
print(index_result)  # 1
```
{% endraw %}

## 3. 딕셔너리 메서드


**`get()`** : 키에 해당하는 값을 반환, 키가 없으면 기본값을 반환


{% raw %}
```python
info = {"name": "Kim", "age": 25}
result1 = info.get("name")
result2 = info.get("address", "Unknown")
print(result1)  # "Kim"
print(result2)  # "Unknown"
```
{% endraw %}


**`keys()`** : 딕셔너리의 모든 키를 반환


{% raw %}
```python
data = {"x": 10, "y": 20, "z": 30}
keys = list(data.keys())
print(keys)  # ["x", "y", "z"]
```
{% endraw %}


**`values()`** : 딕셔너리의 모든 값을 반환


{% raw %}
```python
data = {"x": 10, "y": 20, "z": 30}
values = list(data.values())
print(values)  # [10, 20, 30]
```
{% endraw %}
