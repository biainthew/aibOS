### 자료형 종류


**숫자형**


`int` : 정수


`float` : 실수


`complex` : 복소수


**문자열**


`str` : 문자열(자바 `String`과 유사하지만 불변)


작은따옴표 또는 큰따옴표 사용 가능


**목록**


`list` : 리스트 `[1, 2, 3]` - 자바 `ArrayList` 와 유사, 가변


`tuple` : 튜플 `(1, 2, 3)` - 리스트와 비슷하지만 불변


`range` : 범위 `range(10)` - 주로 반복문에서 사용


`set` : 집합 `{1, 2, 3}` - 자바의 `HashSet`과 유사, 중복 불가, 순서 없음


`dict` : 딕셔너리 : `{key : value}` - 자바의 `HashMap`과 유사, 키-값 쌍


### 자바와의 차이점

1. 파이썬은 변수 선언 시 타입을 명시하지 않음
2. 불변, 가변 개념이 중요
3. 리스트는 서로 다른 타입을 섞어서 저장 가능

# 4. 타입 유추하기
memo = {}
def func(n):
    if n in memo:
        return memo[n]
   
    if n <= 2:
        memo[n] = 1
    else:
        memo[n] = func(n - 1) + func(n - 2)
        
    print(n, memo[n])
    return memo[n]
    
result = func(5)
print("결과:", result)

# 처음 코드 봤을 때 memo는 중괄호로 선언되어 있음
# → 중괄호는 set 아니면 dict
# → key 값이 존재하면 dict 아니면 set 이라고 판단해야 함

### print() 함수 구분자

1. **기본 구분자는 공백**

{% raw %}
```python
print(1, 2, 3)
# 출력: 1 2 3
```
{% endraw %}

1. **sep 파라미터로 구분자 변경 가능**

{% raw %}
```python
print(1, 2, 3, sep=', ')
# 출력: 1, 2, 3

print(1, 2, 3, sep='')
# 출력: 123

print(1, 2, 3, sep='\n')
# 출력:
# 1
# 2
# 3
```
{% endraw %}

1. **end 파라미터로 마지막 문자 변경(기본값은 줄바꿈)**

{% raw %}
```python
print("Hello", end='')
print("World")
# 출력: HelloWorld

print("Hello", end=' ')
print("World")
# 출력: Hello World

print("줄바꿈 안함", end='')
# 기본값은 end='\n' (줄바꿈)
```
{% endraw %}

1. **응용**

{% raw %}
```python
print(1, 2, 3, sep='-', end='!\n')
# 출력: 1-2-3!
```
{% endraw %}
