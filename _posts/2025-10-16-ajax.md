# _ajax_


: Asynchronous JavaScript and XML / 비동기식 자바스크립트 & XML


: 백그라운드 영역에서 서버와 통신하여, 그 결과를 웹페이지의 일부분에만 표시할 수 있음


: 서버와 JSON, XML, HTML, 텍스트 파일 등을 주고받을 수 있음


### ✔️ 장점


1️⃣ 웹 페이지 전체를 다시 로딩하지 않고도 웹페이지의 일부분만을 갱신 할 수 있음


2️⃣ 웹 페이지가 로드 된 후에 서버로 데이터 요청을 보낼 수 있음


3️⃣ 웹 페이지가 로드 된 후에 서버로부터 데이터를 받을 수 있음


4️⃣ 백그라운드 영역에서 서버로 데이터를 보낼 수 있음


### ✔️ 한계


1️⃣ 클라이언트가 서버에 데이터를 요청하는 [client pull](https://en.wikipedia.org/wiki/Pull_technology) 방식을 사용하므로, 서버 푸시 방식의 실시간 서비스는 만들 수 없음


2️⃣ [바이너리 데이터](https://frhyme.github.io/python/binary_data/)를 보내거나 받을 수 없음


3️⃣ AJAX script 가 포함 된 서버가 아닌 다른 서버로 요청을 보낼 수 없음


4️⃣ 클라이언트의 PC 로 AJAX 요청을 보낼 수는 없음


### ✨ 예제


```javascript
<h1>load1</h1>
<div id="newsWrap_1"></div>
<h1>load2</h1>
<div id="newsWrap_2"></div>

<script>
     $('#newsWrap_1').load('index_ajax.html #news_1')
     $('#newsWrap_2').load('index_ajax.html #news_2')
</script>
```
