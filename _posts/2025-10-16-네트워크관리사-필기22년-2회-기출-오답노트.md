### IPv4 의 IP Address 할당에 대한 설명으로 옳지 않은 것은 ?

: 14개


### ARP 에 대한 설명으로 올바른 것은 ?


: ARP 를 이용하여 IP 가 중복되어 사용되는지 찾을 수 있다


### 포트번호로 옳지 않은 것은 ?


: 23번 FTP


### TFTP 옳지 않은 것은 ?


: 3방향 핸드셰이킹 방법인 TCP 세션을 통해 전송한다


### UDP 패킷의 헤더에 속하지 않는 것은 ?


: Window


### IP 데이터그램 헤더 구조의 Field Name 으로 옳지 않은 것은 ?


: Port Number


### OSI 7 Layer 의 전송계층에서 동작하는 프로토콜들만으로 구성된 것은 ?


: TCP, U


### IEEE 802 프로토콜의 연결이 올바른 것은 ?


: IEEE 802.11 - 무선 랜


### 하나의 기기와 서비스에 모든 정보통신 기술을 묶은 새로운 형태의 융합 상품을 말하는 것으로서 크게 유선과 무선의 통합 통신과 방송의 융합 온라인과 오프라인의 결합 등을 말한다


: 디지털 컨버전스


### OSI 7 Layer 중 세션계층의 역할로 옳지 않은 것은 ?


: 에러 제어


### Data 의 저장이 매우 중요해진 요즘에 대용량의 저장장치를 구매하려면 많은 비용이 발생한다 A 를 이용하면 PC 급으로도 파일 서버를 구축할 수도 있고 비용 또한 저렴하다 다수의 접속자를 허용하게 하여 동시에 파일을 잃기 쓰기 또한 간편하며 장비만 있으면 바로 동작하기 때문에 공간 활용도 좋다


: NAS


### L 사용자에 대한 패스워드의 만료기간 및 시간 정보를 변경하는 명령어는 ?


: chage


### L 에서 DNS 의 SOA 레코드에 대한 설명으로 옳지 않은 것은 ?


: TTL 값이 길면 DNS 의 부하가 늘어난다


### L 시스템에서 특정 파일의 권한이 -rwxr-x-x 이다 이 파일에 대한 설명 중 옳지 않은 것은 ?


: 동일한 그룹에 속한 사용자는 실행 권한만을 갖는다


### L 에서 기본적으로 시스템 설정 파일이 위치하는 디렉터리는 ?


: etc


### L 시스템 명령어중 root 만 사용 가능한 명령은 ?


: chown


### L 에서 프로세스와 관련된 명령어에 대한 설명 중 옳지 않은 것은 ?


: top - 가장 우선순위가 높은 프로세스를 보여주는 명령어


### W 에서 폴더에 저장할 수 있는 용량을 제한하고 특정한 파일의 유형은 업로드 하지 못하도록 설정하고자 한다 이러한 설정을 통해서 서버 담당자는 좀 더 유연하고 안전한 파일 서버를 구축할 수 있게 된다 다음 중 다당자가 구축해야 할 적절한 서비스는 무엇인가 ?


: FSRM File Server Resource Manager


### 공통으로 사용되는 저장소를 중앙에서 관리함으로써 각각의 컴퓨터에 저장소를 가지고 있을 때보다 여유 공간의 활용도가 높으며 대규모 이상의 환경에서 주로 구성되고 있다 일반적으로 파이버 채널 연결을 이용하여 데이터 접근이 빠르며 대용량 블록 기반의 데이터 전송 기능으로 LAN 에 독립적인 데이터 백업 복구에 탁월한 기능이 있다 


: SAN Storage Area Network


### W 가 설치된 컴퓨터는 항상 가동하는 것이 일반적인 용도이기 때문에 시스템을 종료할 때마다 그 이유를 명확히 하여 시스템을 좀 더 안정적으로 운영하고자 한다 다음 중 이전에 종료 또는 재부팅된 기록을 알 수 있는 항목은 ?


: 이벤트 뷰어


### FTP 서버에서 따로 포트 대역을 설정해주고 서버는 임의로 지정된 데이터 포트정보를 클라이언트에 보내 클라이언트에서 해당 포트로 접속하는 방식은 ?


: Passive Mode


### W 이 설치된 서버의 용량이 부족하여 4TB 하드디스크를 추가하였다 올바른 파티션 형식은 ?


: GPT (GUID Partition Table)


### OSPF 프로토콜에 대한 설명으로 옳지 않은 것은 ?


: 라우터 사이에 서로 인증하는 것이 가능하여 관리자의 허가 없이 라우터에 쉽게 접속하고 네트워크를 확장할 수 있다


### 255.255.255.224 인 서브넷에 최대 할당 가능한 호스트 수는 ?


: 30개


### IP 헤더에 포함이 되지 않는 필드는 


: ACK


### IPv6 의 주소 표기법으로 올바른 것은 ?


: 3ffe:1900:4545:0003:0200:f8ff:ffff:1105

: FIN


### ICMP 메시지의 타입번호와 설명으로 옳지 않은 것은 ?


: 타입 0 Echo Request


### SSH Protocol 의 특징을 설명한 것 중 올바른 것은 ?

### 네트워크 주소가 192.168.100.128 이며 서브넷 마스크가 255.255.255.102 인 네트워크가 있다 이 네트워크에서 사용 가능한 마지막 IP 주소는 무엇인가 ?


: 192.168.100.190


### OSI 7 Layer 에서 데이터링크 계층의 기능으로 옳지 않은 것은 ?


: Text 의 압축 암호화 기능


### 에러제어 기법 중 자동 재전송 기법으로 옳지 않은 것은 ?


: 전진에러 수정 (FEC)


### 100BASE-T 라고도 불리는 이더넷의 고속 버전으로서 100Mbps 의 전송속도를 지원하는 근거리 통신망의 표준은 ?


: Fast Ethernet


### 늘어나는 서버 관리 업무에 스트레스를 받음 서버가 대량으로 추가되어 서버실에 놓을 공간도 모자랄 뿐만 아니라 전기 용량 문제로 시설과의 협의중이나 어려움이 예상되고 있다 또한 서버들에게서 발생하는 발열 문제로 24시간 냉각장치 및 항온항습으로 인한 발생비용 또한 회사에서 줄여보라고 지시가 내려와서 비용도 많이 발생 외주의 필요성이 있는 듯 고민 끝에 이것을 이용하여 전용회선 및 안정적인 전력공급을 받기로 했다


: IDC (Internet Data Center)


### 유선망과 무선망을 통합하여 통합 단말기를 통해 최적으로 통신망 간 접속이 이루어지며 끊김없는 광대역 멀티미디어 서비스를 제공한다 특정 통신망이나 단말기에 구애받지 않고 통신 방송 ㅁ및 인터넷 서비스를 통합하며 서비스 품질보장 광대역화 고기능화 보안 보장 및 IPv6 지원 등의 특성이 있다


: 광대역 융합망


### W 에서 파일 및 프린터 서버를 사용할 수 있도록 지원하기 위해서 반드시 설치해야 하는 통신 프로토콜은 ?

### L 시스템에서 데몬에 관한 설명 중 옳지 않은 것은 ?


: 시스템 부팅 때만 시작될 수 있다


### 다른 운영체제와 L 가 공존하는 하나의 시스템에서 멀티 부팅을 지원할 때 사용되며 L 로더를 의미하는 것은 ?
