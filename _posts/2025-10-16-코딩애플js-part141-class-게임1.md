### 2D 게임의 동작원리


chrome dinosaur game 치면 나오는 2D 플랫폼 게임을 만들 예정

1. 빨간 네모를 여러개 배치해서 사물을 만들고 초록 네모를 배치해서 내 캐릭터 만들기
2. 그 다음에 방향키를 입력하면 네모가 어떻게 움직일지 코드 짜고
3. 네모끼리 충돌하는지 .1초마다(아니면 1프레임마다) 계속 체크해서 충돌시 원하는 작업을 해주면 게임개발 끝임

### 비슷한 obj들 많이 필요


게임을 잘 생각해보면 비슷한 사물이 많이 등장하는데


이런 사물이 어디서 생성될지 사이즈는 어떻게 되는지 어떤 속도로 움직이는지 같은 정보들을 object 아니면 array 같은 곳에 담아두면 좋지 않을까 ?


```javascript
var 장애물1 = { x : 0, y : 0, 속도 : 100, 사이즈 : '10px' }
var 장애물2 = { x : 10, y : 10, 속도 : 100, 사이즈 : '10px' }
var 장애물3 = { x : 0, y : 0, 속도 : 100, 사이즈 : '20px' }
...
```


그리고 이걸 필요한 만큼 반복 생성해야한다.


그래서 비슷한 object 많이 만들 때는 class 를 쓰면 편하다
