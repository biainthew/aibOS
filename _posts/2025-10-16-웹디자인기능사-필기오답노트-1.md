### 조건등색(메타메리즘)


: 두 가지 다른 색이 특정 광원에서 하나의 색으로 보이는 것


### 연색성


: 광원에 따라 물체의 색상이 다르게 보이는 현상


### 색료의 삼원색


: 청록, 자주, 노랑


### 신인상파 화가 쇠라의 ‘점묘화’ 는 무슨 혼합의 예인가


: 병치혼합 (점묘화와 유사한 TV 화소처럼 가까이서 볼 때는 서로 다른 색상의 점이지만, 멀리서 볼 때 하나의 색으로 보이는 현상)


### 건물 내 비상구 표시나 계단의 비상 표시등을 어두운 곳에서도 볼 수 있게 초록으로 하는 것은 무슨 현상을 응용한 것인가


: 푸르킨예 현상 (어두운 곳에서 푸른 것이 더 밝게 느껴지는 현상)


### CIP(Corporate Identity Program) 의 분류 중에서 기본 요소에 해당되는 것은


: CIP 란 기업이나 공공단체가 갖고 있는 이미지를 시각적으로 체계화, 단일화 한 것으로 기업의 실체를 확실하게 인식시키며 질서 있고 체계적인 관리를 통한 합리적인 기업경영에 목적이 있는 것 / 정답은 전용 색상


### 선에 대한 설명으로 틀린 것


: 곡선은 우아, 매력, 모호, 유연, 섬세함과 ~~정적인~~ 표정을 나타낸다 → 동적


### 인간을 중심으로 한 디자인 작업 시 고려해야 할 기능성과 거리가 먼 것


: 시장적 기능


### TCP/IP


: TCP 는 OSI 의 전송 계층에 해당한다 / 이메일 이용 시 필요하다 / 인터넷에서 모든 시스템들이 이 프로토콜을 이용한다


### IPv4 주소에서 각 클래스별 첫 8bit 값이 옳은 것


 : A : 0000 0001 ~ 0111 1110 / B : 1000 0000 ~ 1011 1111


### 웹페이지 제작과정


: 스토리보드 작성 → 전체구조, 페이지수, 메뉴결정 → 전체 홈페이지 스케치
