### 스레드


프로그램이 실행되는 흐름의 단위로 한 프로세스 안에서 여러 작업을 동시에 실행할 수 있도록 하는 매커니즘이다 자바와 같은 언어에서 스레드를 사용하면 여러 작업을 병렬로 처리할 수 있어 프로그램의 성능을 높이거나 동시에 여러 작업을 수행할 수 있다


**스레드와 프로세스의 차이**

- 프로세스 : 운영체제에서 실행 중인 프로그램의 인스턴스를 말한다 각 프로세스는 독립된 메모리 공간을 가지고 서로 간섭하지 않는다
- 스레드 : 하나의 프로세스 내에서 동시에 실행될 수 있는 흐름이다 한 프로세스는 여러 스레드를 가질 수 있으며 이 스레드들은 프로세스의 공유 메모리를 사용한다

### 스레드 안전(Thread Safety)


스레드 안전이란 여러 스레드가 동시에 접근해도 문제가 발생하지 않도록 보호된 상태를 말한다


멀티 스레딩 환경에서 두 개 이상의 스레드가 동일한 객체나 메모리 공간에 동시에 접근할 때 해당 객체가 올바르게 동작하고 데이터 일관성이 유지되는 것을 보장해야 한다
