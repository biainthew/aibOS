### 이더넷에서 헤더 부분을 포함한 최대 데이터 크기로 가장 올바른 것은 ?


: 1500Byte


### 다음 중 DHCP 클라이언트가 자기가 갖고 있던 IP Addresss 와 TCP/IP 관련 설정을 모두 버리고 초기화하는 명령으로 올바른 것은 ?


: ipconfig/release


### IP Address 190.1.0.0 가 속한 주소로 가장 올바른 것은 ?


: 네트워크 ID 주소


### IP Address 에서 Host ID 의 모든 비트가 0인 경우는 ?


: 자신의 망


### IP Address 가 B Class 이고 전체를 하나의 네트워크로 사용할 때 적절한 서브넷 마스크 값은 ?


: 255.255.0.0


### RIP 프로토콜의 일반적인 특징을 기술한 것으로 옳지 않은 것은 ?


: 멀티캐스팅을 지원한다


### IGRP(Internet Gateway Routing Protocol) 의 특징으로 옳지 않은 것은 ?


: 링크 상태 프로토콜로 메트릭의 비용을 이용하여 라우팅 최적화를 한다


### Internet Protocol 에 대한 설명으로 옳지 않은 것은 ?


: OSI 7 Layer 의 Datalink Layer 에 해당된다


### TCP 에 대한 설명으로 옳지 않은 것은 ?


: 비연결 지향형이다


### Tocken Ring 이 사용하는 디지털 신호 코딩은 ?


: Differential Manchester


### 다음 중 광통신에 대한 설명으로 옳지 않은 것은 ?


: 기존의 선로보다 무게가 무겁지만 부피는 작다


### 전송 데이터의 헤더에 메시지 수신 상태 등의 제어 정보, 전송할 데이터 수 등의 동기 문자 정보를 포함시켜 전송하는 Byte 방식의 프로토콜은 ?


: DDCMP


### HDLC 의 프레임에서 플래그 필드를 2진수로 표현한 것은 ?


: 01111110


### 다음 중 유선통신 전송 매체로 옳지 않은 것은 ?


: Micro Wave


### RAID 의 기능 중에서 Hot Swap 의 기능을 가정 올바르게 설명한 것은 ?


: 전원이 켜진 상태에서 디스크를 교체하는 기능


### 무선 랜카드를 설치한 5대 이하의 PC 를 엑세스 포인트 없이 서로간에 통신이 가능하도록 구성할 때 사용하는 모드는 ?


: 애드 혹(Ad-Hox) 모드


### Dummy Hub 에 대한 설명 중 가장 올바른 것은 ?


: Hub 에 IP Address 를 배정하지 않아도 통신에는 지장이 없다


### 파일 시스템에 관한 설명 중 가장 옳지 않은 것은 ?


: FAT  16/32  파일 시스템을 NTFS 로 변환하면 모든 자료는 포맷된다


### W 에서 IIS 웹 서버의 설정 중 하루에 평균적으로 방문하는 사용자 수를 설정하기 위해 사용하는 탭은 ?


: 성능 탭


### W 의 IIS 보안 통신 기능을 이용하기 위해서는 유효한 서비스 인증 도구를 먼저 설치해야 한다 사용자들이 웹사이트에 사용하는 보안통신은 ?


: SSL


### W 의 IIS 기본 사양으로 설치 시 포함되지 않은 것은 ?


: FTP 서버


### W 에서 DNS 서버 종류로 옳지 않은 것은 ?


: Expert Server


### 다음 중 W 에서 FTP 사이트 등록정보의 메시지 탭에서 입력할 수 있는 내용으로 옳지 않은 것은 ?


: 거부된 IP Address 나 도메인 이름에 보내는 메시지를 입력할 수 있다


### W 에서 DHCP 서버 서비스를 설치한 후 주소 범위를 설정하려고 한다 이때 주소 범위 설정에 사용되지 않는 옵션은 ?


: DHCP 서버의 MAC 주소


### TCP/IP 주소 관리를 간소화하기 위하여 자동으로 IP Address 를 할당하고자 할 때 사용하는 서비스는 ?


: DHCP


### 터미널 서버의 관리 툴인 터미널 서비스 구성 메뉴에서 할 수 있는 일로 옳지 않은 것은 ?


: 연결된 사용자를 확인하고 사용자에게 메시지를 보낸다


### W 의 새 사용자 만들기 옵션에 대한 설명으로 옳지 않은 것은 ?


: 암호사용기간 제한없음 : 주로 사용자 계정에서 사용하며 해당 사용자만이 암호를 변경할 수 있다


### SQL Server 를 관리하는데 있어서 편리한 사용자 인터페이스를 제공하는 툴로 트랜잭트 SQL 문을 모르더라도 데이터베이스나 테이블을 생성하는 작업을 손쉽게 할 수 있으며 데이터베이스 백업 및 복구 작업을 할 수 있는 것은 ?


: 엔터프라이즈 매니저


### W 에서 Disk Quota 에 대한 설명으로 가장 올바른 것은 ?


: User 가 특정 파일 시스템에서 사용할 수 있는 디스크의 양을 제한한다


### W 에서 대규모 파일 및 분할지원 실시간 압축 기능 신뢰성 등을 제공하는 파일 시스템은 ?


: NTFS


### Linux 시스템에서 Is 라는 명령어 사용법을 알아보는 명령어로 올바른 것은 ?


: man Is


### Linux 시스템에서 운용체계의 실제 기능들을 제공하는 부분으로 DOS 의 msdos.sys, io.sys 에 해당하는 부분은 ?


: Kernel


### Linux 시스템을 설치하고 부팅하였더니 li 메시지만 나오고 시스템이 멈추었다 이때 LILO 를 삭제하는 방법은 ?


: fdisk/mbr 명령어를 사용하여 부트 레코드를 제거한다


### 목적지 노드까지의 연결 경로를 알기 위해 사용하는 유틸리티는 ?


: Tracert


### Ipv4 의 IP Address 체계는 몇 개의 옥텍트로 구성되어 있는가 ?


: 4개


### 국가 도메인은 각 대륙별 나라별로 등록 기구를 두고 관리하고 있다 우리나라가 속해있는 대륙별 등록기구는 ?


: APNIC


### TCP/IP 의 기본설정 DNS 서버 주소를 확인할 수 있는 명령어로 올바른 것은 ?


: ipconfig/all


### 라우팅 프로토콜은 내부 라우팅 프로토콜과 외부 라우팅 프로토콜로 구분된다 다음 중 외부 라우팅 프로토콜로 올바른 것은 ?


: BGP


### TCP/IP 에서 Broadcast 의 의미로 올바른 것은 ?


: 한 호스트가 로컬 랜 상의 모든 호스트에게 메시지를 전송하는 것


### ICMPv6 에서 IPv4 의 ARP 역할 및 특정 호스트로의 전달 가능 여부 검사 기능을 하는 메시지는 ?


: 이웃 요청과 광고 메시지


### IP 패킷은 네트워크 유형에 따라 전송량에 있어 차이가 나기 때문에 적당한 크기로 분할하게 된다 이때 기준이 되는 것은 ?


: MTU


### TCP 에 대한 설명으로 올바른 것은 ?


: 수신 순서는 송신 순서와 동일하다


### TCP 를 직접 이용하는 응용 프로그램으로만 묶인 것은 ?


: Telnet, SMTP


### Annonymous FTP 에 대한 설명으로 가장 옳지 않은 것은 ?


: Annonymous FTP 를 사용하면 텍스트 문서만을 송수신할 수 있다


### Macintosh 와 IBM 호환 기종간 네트워크 연결 작업을 하려고 할 때 주로 사용하는 프로토콜은 ?


: Apple Talk


### 전이중 방식에서 사용할 수 없는 프로토콜은 ?


: BSC


### 이더넷에서 사용하는 매체 엑세스 방식은 ?


: CSMA/CD

: 물리 - 데이터 - 네트워크 - 전송 - 세션 - 프리젠테이션 - 응용


### IEEE 802 표준에 대한 연결이 잘못 이루어진 것은 ?

### W 의 IIS 에서 설정 가능한 서비스로 옳지 않은 것은 ?


: TELNET 서비스


### W 설치 도중에 파티션을 어떤 File System 으로 포맷할 것인지를 결정해야 한다 W 에서 지원하지 않는 파티션은 ?

### DNS 서버에 IP Address 를 도메인네임으로 변환하는 기능을 가지도록 할 때 추가해야 하는 영역은 ?


: 역방향 조회 영역


### IIS 로 가상 FTP 사이트를 구성할 경우에 대한 설명으로 옳지 않은 것은 ?


: 하나의 IP 로 가상 FTP 사이트를 구성할 수 있지만 서로 다른 도메인 서비스는 구성할 수 없다


### W 의 DHCP 구성 시 같은 IP Address 를 사용하기 위해 IP Address 를 예약한다 이때 설정되는 사항으로 옳지 않은것은 ?


: 서브넷 마스크


### W 에서 사용되는 사용자 계정에 대한 설명 중 올바른 것은 ?


: Administrator 계정으로 로그인 하면 해당 컴퓨터에 대한 모든 권리를 행사할 수 있다


### Linux 시스템의 명령어에 대한 설명으로 옳지 않은것은 ?
