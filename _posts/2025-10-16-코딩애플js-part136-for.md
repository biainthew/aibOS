### 반복문의 종류

1. 그냥 for 반복문 - 어려움
2. forEach() 반복문 - Array 전용
3. for in 반복문 - Object 전용
4. for of 반복문 - iterable 전용

### 반복문의 용도

1. 코드 단순 반복
2. 자료형에 담긴 자료들을 하나씩 꺼내 쓸 때

### for in - Object


: Object 자료형에 저장된 자료들을 하나씩 꺼내고 싶을 때 사용합니다.


```javascript
let 오브젝트 = { name : 'Kim', age : 30 };

for (let key in 오브젝트) {
	console.log(오브젝트[key]);
}

//Kim 30
```

1. 여기서 `key` 는 변수명, 마음대로 작명하면 됨
2. `오브젝트`라는 자료 내부 데이터 개수만큼 반복
3. 반복할 때마다 `key` 라는 변수는 `name`, `age` 이렇게 데이터의 key 값이 됨
4. 반복시마다 변경되는 key 값을 이용하면 오브젝트 내의 자료를 모두 출력할 수 있음
5. 단순히 `오브젝트.key` 를 콘솔창에 출력하면 되는데 변수명을 점 찍고 쓸 수 없으니 `오브젝트[key]` 라고 쓰면 됨

### for in 특징 1. enumerable 한 것만 출력 가능


```javascript
console.log(Object.getOwnPropertyDscriptor(오브젝트, 'name'));
```


이렇게 치면


```javascript
{value: "", writable: true, enumerable: true, configurable: true}
```


이게 나오는데 이게 오브젝트와 함께 저장되는 속성들임 (그래서 오브젝트 자료형이 좀 무거움)


`enumerable`이라는게 있는데 이게 `true` 인 자료들만 for in 반복문이 출력할 수 있음


이걸 강제로 `false` 로 만들면 for in 반복문이 거름


> ❓ **enumerable** ?  
> 셀 수 있는 이라는 뜻


### for in 특징 2. 부모의 prototype 에 저장된 것도 출력해줌


: object 의 부모의 유전자에 있는 것도 반복문으로 출력해줌


```javascript
class 부모 {

}
부모.prototype.name = 'Park';

let 오브젝트 = new 부모();

for(let key in 오브젝트) {
	console.log(오브젝트[key])
}
```


: `Park` 이라는 자료는 부모가 가지고 있는 것인데도 출력해줌 이게 단점 싫다면 ?


```javascript
class 부모 {
 
}
부모.prototype.name = 'Park';

var 오브젝트 = new 부모();

for (var key in 오브젝트) {
  if (오브젝트.hasOwnProperty(key)) {
    console.log(오브젝트[key]);
  }
}
```


: 이런식으로 if 문을 추가해주면 됨


`오브젝트.hasOwnProperty()` 함수는 오브젝트가 이 key값을 직접 가지고 있냐 라고 물어보는 함수


있 `true` 없 `false` 내가 가진 것만 반복시키고 싶으면 이걸 꼭 써야 함


### for of - iterable


```javascript
let 어레이 = [2,3,4,5];
for (let 자료 of 어레이) {
	console.log(자료);
}
```


: array, 문자, arguments, Nodelist, Map, Set 에 전부 다 적용 가능함


근데 정확히 말하면 iterable 인 자료형들에만 적용가능한 반복문


> ❓ **iterable** ?  
> `[Symbol.iterable]()` 이라는 일종의 메소드를 가지고 있는 자료형을 뜻함


이걸 가지고 있으면 for of 반복문을 쓸 수 있다고 알면 됨


document.getElementByClassName() 이나 document.querySelectorAll() 이런 요소로 찾은 요소들은 [] 대괄호 안에 담겨서 오는데 array 는 아니고 NodeList 라고 함


Nodelist 안에 있는 HTML 요소들을 하나씩 꺼내서 처리할 때 매우 자주 쓸 수 있는 반복문이라고 보면 됨

Q2. 어떤 놈이 object 자료의 key 값에 오타를 섞어놨습니다. key 값 마지막 문자에 한자릿수 숫자가 섞여 있으면 그걸 다 제거하고 싶습니다. 어떻게 코드를 짜면 될까요 ? 마지막 글자가 숫자인지 판단하는 방법도 잘 찾아보셈


```javascript
var products = [
  {
    name1 : 'chair',
    price1 : 7000,
  },
  {
    name2 : 'sofa',
    price : 5000,
  },
  {
    name1 : 'desk',
    price3 : 9000,
  },
];
```


내가 쓴 답


```javascript
for (let key of products) {
	개망
}
```


정답 코드


```javascript
let newValue;
let newKey;

for (let item of products) {
    for (let key in item) {
        if (isNaN(parseInt(key.slice(-1))) == false) {
            newValue = item[key];
            newKey = key.slice(0, -1);
            item[newKey] = newValue;
            delete item[key]
        }
    }
}
console.log(products);
```
