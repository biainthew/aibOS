| en                         | 사용자모드에서 관리자모드로 전환     |
| -------------------------- | --------------------- |
| conf t                     | 관리자모드에서 전역설정모드로 전환    |
| interface fastethernet 0/0 | fastethernet 으로 이동    |
| interface serial 2/0       | serial 로 이동           |
| ip add                     | ip 설정                 |
| no shutdown                | 활성화                   |
| bandwidth 2048             | 대역폭을 2048로 설정         |
| clock rate 72000           | 클럭 속도를 72k 로 설정       |
| description ICQA           | 주석을 ICQA 로 설정         |
| ip add … secondary         | 두 번째 ip 주소 설정         |
| ip default-gateway …       | 기본 게이트웨이 ip 설정        |
| ip dhcp pool icqa          | dhcp 서버 이름을 icqa 로 설정 |
| network …                  | dhcp 서버 ip 주소 설정      |
| line vty 0 4               | 텔넷 설정 명령어             |
| line console 0             | console 0 으로 이동       |
| password icqa              | 비밀번호 icqa 로 설정        |
| login                      | 로그인                   |
| exec-timeout 03 50         | 텔넷에서 입력이 없으면 세션 자동종료  |
| hostname network2          | 호스트네임을 network2 로 설정  |
| show interface             | 인터페이스 정보확인            |
| show user                  | 사용자 정보확인              |
| show ip route              | 라우팅 테이블 정보확인          |
| show flash                 | 플래시 내용 확인             |
| show process               | 프로세스 정보 확인            |
