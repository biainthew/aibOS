### Cookie 객체

1. name : 쿠키의 이름, 필수 속성, 쿠키를 식별할 때 사용
2. value : 쿠키에 저장된 값, 주로 사용자 정보를 암호화 하거나 세션 ID와 같은 식별 값을 저장하는 데 사용
3. maxAge : 쿠키의 유효 기간(초 단위), 양수일 경우 설정한 시간 후에 쿠키가 만료되고 0일 경우 쿠키가 즉시 삭제됨, 음수는 브라우저 종료 시 쿠키가 삭제되는 세션 쿠키를 의미함
4. path : 쿠키가 유효한 경로를 설정, 이 경로로 시작하는 URL에만 쿠키가 전송됨, 예를 들어 `"/app"` 으로 설정하면 `/app/*` 경로에만 쿠키가 유효하게 전송됨
5. secure : true 로 설정하면 HTTPS 연결에서만 쿠키가 전송됨, 기본값은 false
6. httpOnly : true 로 설정하면 자바스크립트로 쿠키에 접근할 수 없음, XSS 공격으로부터 쿠키를 보호하는 데 유용
7. comment : 쿠키에 대한 설명을 추가하는 속성, 현재는 거의 사용되지 않으며 대부분의 브라우저가 무시

**예시 :** `request.getCookies()` 로 얻은 쿠키 배열의 첫 번째 쿠키 객체에서 각 속성에 접근하려면 ?


```java
Cookie[] cookies = request.getCookies();
if (cookies != null) {
    for (Cookie cookie : cookies) {
        System.out.println("Name: " + cookie.getName());
        System.out.println("Value: " + cookie.getValue());
        System.out.println("Max Age: " + cookie.getMaxAge());
        System.out.println("Path: " + cookie.getPath());
        System.out.println("Domain: " + cookie.getDomain());
        System.out.println("Secure: " + cookie.getSecure());
        System.out.println("HttpOnly: " + cookie.isHttpOnly());
    }
}
```
