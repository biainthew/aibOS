### **1. 리액트에서 레이아웃 만들 때 쓰는 JSX 문법 3개**

1. html에 class 넣을 땐 class 대신 className

    ```javascript
    function App(){
      return (
        <div className="App">
          <div className="black-nav">
            <h4>블로그임</h4>
          </div>
        </div>
      )
    }
    ```

2. 변수를 html에 꽂아넣을 때는 { 중괄호 } = **데이터 바인딩**

    ```javascript
    function App(){
    
      let post = '강남 우동 맛집';
      return (
        <div className="App">
          <div className="black-nav">
            <div>블로그임</div>
            <div>{ post }</div>
          </div>
        </div>
      )
    }
    ```

3. html 에 style 속성 넣고 싶으면 { { 이 안에 } }

    ```javascript
    <div style={ {color : 'blue', fontSize : '30px'} }> 글씨 </div>
    ```
