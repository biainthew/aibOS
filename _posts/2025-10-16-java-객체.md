자바에서 객체(Object)란 클래스(Class)라는 틀을 바탕으로 생성된 인스턴스이다 객체는 프로그램에서 특정 데이터를 가지며 그 데이터를 조작할 수 있는 메서드를 포함하고 있다


객체는 상태(필드)와 행동(메서드)를 가진 독립적인 단위이다


예를 들어 자동차 클래스를 만든다고 가정했을 때 자동차 객체는 색상, 모델, 속도와 같은 속성(필드)을 가지고 가속, 정지와 같은 행동(메서드)을 수행할 수 있다


### 특징

1. 캡슐화 : 객체는 데이터를 숨기고 메서드를 통해서만 접근하도록 함으로써 데이터 무결성을 보호한다
2. 상속 : 객체는 상위 클래스의 속성과 메서드를 상속받아 코드 재사용성과 구조화를 향상시킨다
3. 다형성 : 객체는 동일한 인터페이스 또는 상위 클래스의 메서드를 각기 다른 방식으로 구현하여 유연한 코딩을 가능하게 한다
4. 추상화 : 객체는 복잡한 데이터를 감추고 꼭 필요한 부분만을 외부에 노출하여 간결성을 제공한다

### 생성 예시


```java
public class Car {
	String color;
	String model;
	
	public void drive() {
		System.out.println("The car is driving")
	}
}

// 객체 생성
Car myCar = new Car();
myCar.color = "red";
myCar.model = "Sedan";
myCar.drive();
```


이처럼 자바 객체는 특정 클래스의 인스턴스로 프로그램내에서 실제 데이터를 담고 행동을 수행할 수 있다.
