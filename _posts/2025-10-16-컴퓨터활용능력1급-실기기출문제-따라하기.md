### 1. DB 구축

1. 확대/ 축소 창 : shift + f2
2. 입력마스크 68-2956 처럼 여섯자리 숫자, -도 저장되도록

    ```sql
    00-0000;0;
    ```

3. 반드시 한 글자만 입력되도록 유효성 검사 규칙 설정

    ```sql
    len([필드명])=1
    ```

4. 기본키가 아니면서도 중복된 값이 입력되지 않도록 설정
: 인덱스 → 예(중복 불가능)
5. A 테이블의 내용을 B 테이블에 추가하기

    만들기 → 쿼리추가


    교번 필드를 이용하여 중복여부 판단하기


    ```sql
    not in (select 교번 from 추가된 명단)
    ```

6. 관계 추가

    1:M 관계일 때 1인 애를 M인 애로 드래그


    기본키 설정은 마우스 오른쪽, 기본키

7. 폼 컨트롤에 전체 인원수 표시되도록, 인원이 없어도 0으로 표시되는 기호 사용

    속성시트 → 모두 → 컨트롤 원본 → `=Count(*)`


    속성시트 → 형식 → 형식 → 0명

8. 컨트롤에 부서명(부서번호) 가 표시되도록 설정

    속성시트 → 데이터 → 컨트롤 원본


    ```sql
    [부서명] & "(" & [부서번호] & ")"
    ```

9. 보고서 컨트롤 값이 680302-1025484 처럼 입력되도록 입력마스크 설정

    ```sql
    000000-0000000
    ```

10. 보고서 열기 매크로 작성

    만들기 → 매크로

11. 주민등록번호 필드명을 다르게 표시하고 싶을 때 - 생년월일:주민등록번호
12. 선교부 로 끝나는 것들 선택하기 - 조건에

    ```sql
    *선교부
    ```
