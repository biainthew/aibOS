# _API_


: Application Programming Interface


: 함수의 이름만 보고도 무슨 일을 할 지 예상 가능하게 만들어야 함


### ✔️ 문법


> 💡 `function name (param1, param2 …){실행문 (코드)}`


👉🏻 특정 기능을 수행하도록 작성된 코드 블록


👉🏻 매개변수(parameter) : 값이 아직 정해지지 않은 변수, 필요한 곳에서 값을 대입시켜 실행


### 🌈 예시 -3


> 💡 `function(message){  
>     console.log(message);  
> };  
> log(’hi bia’)   // hi bia`


👉🏻 message 라는 매개변수에 hi bia 라는 값을 대입해서 콘솔창에 띄워줌


# _Object_


: 객체


### 🌈 예시


> 💡 `function changeName(obj){  
>     obj.name=’bia’;  
> };  
> const bi = {name:’bi’};  
> changeName(bi);   // {name:'bi'}.name='bia'  
> console.log(bi);   // object name : 'bia' prototype : object`


👉🏻  obj 라는 매개변수의 이름은 bia


👉🏻  이름은 bi 라는 값을 변수 bi 에 담아둠


👉🏻  매개변수에 bi 라는 값을 넣어서 함수 실행


👉🏻  bi.name=’bi’ >> 이름은 bia type 은 object


### 🌈 예시


> 💡 `function showMessage(message, from){  
>     if(from===undefined){  
>         from='unknown';  
>     }  
>     console.log(`${message} by ${from}`)  
> }  
> function showMessage(message, from='unknown'){  
>     console.log(`${message} by ${from}`)  
> }  
> showMessage(’hi’);   // hi by unknown  
> showMessage(’hi’ , ‘bia’)  // hi by bia`


👉🏻 from 이라는 변수에 값을 넣어주지 않으면 unknown 이 출력

# _Return value_

> 💡 `function sum(a+b){

# _즉시실행 함수_


: 전에 ; 잘 써줘야 함 에러남


: 사용하는 이유 : 초기화 / 한 번의 실행만 필요로 하는 초기화코드 부분에 많이 사용


### 🌈 예시


> 💡 `// 일반함수  
> function hello(){  
>     console.log('its me')  
> }  
> hello();  
>   
> // 즉시실행함수1  
> (function hello(){  
>     console.log(’its me’);  
> })();  
>   
> // 즉시실행함수2  
> (() ⇒ {  
>     console.log(’its me’);  
> })();`


👉🏻 출력값 모두 its me


👉🏻 괄호로 감싸주고 뒤에 괄호 추가 / 괄호로 감싸주고 괄호 화살표 대괄호 뒤에 괄호 추가


👉🏻 사용하는 이유 : 초기화 / 한 번의 실행만 필요로 하는 초기화코드 부분에 많이 사용
