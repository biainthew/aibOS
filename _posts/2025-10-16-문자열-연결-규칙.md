### Java에서는 문자열 연결 시 왼쪽부터 순서대로 처리된다


```java
// 예시1 : 숫자로 시작
int x = 5;
int y = 7;

System.out.println(x + y + "Java");

// 숫자 + 숫자 + 문자열 → 숫자 먼저 계산
→  5 + 7 + "Java"
→  12 + "Java"
→  "12Java"
```


```java
// 예시2 : 문자로 시작
int x = 5;
int y = 7;

System.out.println("Java" + x + y);

// 문자열 + 숫자 + 숫자 → 왼쪽부터 문자열로 연결
→  "Java" + 8 + 15
→  "Java8" + 15
→  "Java815"
```


만약 무조건 숫자를 더해서 출력하고 싶다면


```java
// 이렇게 괄호로 묶어야 함
System.out.println("Java" + (x + y));
```


### 결론


숫자 + 숫자 = 더하기


문자열 + 숫자 = 문자열 연결


문자열 + 문자열 = 문자열 연결
