### UDP 에 대한 설명으로 가장 타당하지 않은 것은 ?


: UDP 가 제공하는 오류검사는 홀수 패리티와 짝수 패리티가 있다


### ARP(Address Resolution Protocol) 에 관한 설명으로 타당한 것은 ?


: IP 주소를 물리계층에서 이용하는 하드웨어 주소로 매핑하는 기능을 제공한다


### TCP 헤더 중에서 에러제어를 위한 필드는 ?


: Checksum


### ICMP 의 메시지 사용에 대한 설명으로 타당하지 않은 것은 ?


: 호스트의 IP 가 중복된 경우


### TCP/IP 응용프로토콜이 아닌 것은 ?


: UDP


### SNMP 의 설명으로 타당하지 않은 것은 ?


: SNMP 는 TCP 세션을 이용한다


### IP 헤더 구조에 대한 설명으로 타당하지 않은 것은 ?


: TTL - 출발시간을 위한 필드


### 텔넷의 기능으로 옳은 것은 ?


: 네트워크를 통한 가상 단말기 기능


### 주로 로컬 디스크가 없는 시스템에서 사용하며 IP 정보를 저장하지 않기 때문에 부팅할 때마다 이 프로토콜을 이용해 IP 주소를 할당하는 과정이 필요하다


: RARP


### IGMP(Internet Group Management Protocol) 에 대한 설명으로 가장 타당한 것은 ?


: 다중 전송을 위한 프로토콜이다


### 숫자로 된 IP 주소를 우리가 친숙한 문자주소 / 인터넷 표준이름으로 바꾸어 주거나 그 반대 역할을 수행한다


: DNS


### IP(155.100.100.5) 가 속하는 Class 는 ?


: Class B


### 프로토콜의 일반적인 기능 중 캡슐화를 할 때 제어정보에 포함되지 않는 것은 ?


: 연결제어 (Connection Control)


### 두 개 이상의 독립적인 네트워크가 서로 연동되어 연결되는 형태의 망은 ?


: Catenet


### OSI 모델에서 계층구조상 기본 구성요소 중 개방형 시스템에서 각 계층별로 정보와 데이터를 전송하는 행위 ?


: 서비스


### 다음의 망 연동 장치 중 네트워크 계층에서 사용하며 여러개의 서브네트워크를 연결할 때 사용하는 것은 ?


: Router


### 발신 가입자로부터 수신자까지의 모든 전송, 교환 과정이 디지털로 처리되며 음성과 비음성 등 서비스를 종합적으로 처리하는 통신망은 ?


: ISDN


### 인터넷 계층적 구조가 순서에 맞게 배치된 것은 ?


: 데이터링크 - 네트워크 - 트랜스포트 - 응용


### OSI 참조모델의 계층별 기능에 대한 설명으로 적절하지 않은 것은 ?


: 데이터 링크 계층 : 응용 프로그램의 인터페이스와 통신을 싱행하기 위한 응용 기능을 제공한다


### 하나의 웹사이트에 디렉토리별로 인증과 권한을 다르게 설정할 경우 사용하는 디렉토리는 ?


: Virtual Directory


### Windows 2000 Server 의 IIS 웹서버를 설치할 경우 자동으로 설치되는 것이 아닌 것은 ?


: POP3


### DNS 서버의 이름을 사용자들이 편하게 사용할 수 있도록 하기 위해서는 DNS 대화상자에서 해당 서버의 어떤 메뉴를 실행하는가 ?


: 새 별칭


### 현재 FTP 사이트에 접속 중인 사용자를 확인할 경우 FTP 서버의 등록정보에서 어느 탭을 클릭하는가 ?


: FTP 사이트


### 다음 중 인터넷 정보 서비스 메뉴에서 관리 웹 사이트 등록정보에 없는 탭은 ?


: LDAP 라우팅


### 다음은 로컬 사용자 계정 만들기에서 새 사용자 대화상자이다 디폴트는 어느 부분에 체크되어 있는가 ?

### Windows 2000 Server 가 설치되면서 자동 생성되는 그룹으로 수정이 불가능한 그룹은 ?


: Built-in Group


### Windows 2000 Server 의 네트워크 방식 중에서 모든 계정과 자원을 특정 서버에서 관리하는 중앙 집중식 방식은 ?


: 도메인 방식


### 계정 잠금에 대한 설명으로 옳은 것은 ?


: 잠금이 발생하기 위한 로그인 실패 횟수는 관리자에 의해 설정될 수 있다


### Linux 에서는 긴 파일명을 허용한다 너무 긴 파일명을 지정할 경우 몇글자만 입력 후 어떤 키를 사용하면 그 이후의 글자가 입력되게 하는가 ?


: Tab 키


### 두 개 이상의 동일한 LAN 사이를 연결하여 네트워크 범위를 확장하고 스테이션 간의 거리를 확장해주는 장치는 ?


: Repeater


### Ethernet 표준 케이블을 사용한 외장 MAU(Medium Access Unit) 를 무엇이라 하는가 ?


: Transceiver


### 최근 공중망 무선랜서비스가 상용화되었는데 가입자는 접속장치에서 100m 반경 이내에 무선 랜카드 장착 노트북 PDA 등을 통해서 서비스를 받을 수 있는 접속장치는?


: AP(Access Point)


### 서로 다른 프로토콜을 가진 두 시스템 간을 연결시키며 OSI 참조모델의 모든 계층에서 동작하는 장치는 ?


: Gateway


### CISCO 사의 2501 라우터를 사용하고 있는 회사에 근무하는 네트워크 관리자이다 지금 라우터의 설정을 바꾸기 위해 라우터에 접속하여야 한다 다음 중 라우터에 접속하기 위해 사용할 수 있는 방법이 아닌 것은 ?


: In-Line 포트를 이용해 연결한다


### Anonymous FTP 에 대한 설명으로 적당하지 않은 것은 ?


: 일반적으로 80번 포트를 사용하여 상호 통신한다


### ICMP 의 Message Type 필드의 유형과 질의 메시지 내용이 잘못 연결된 것은 ?


: 5 - Echo Request : 네트워크상의 두 개 이상의 장비의 기본 연결을 검사하기 위해 사용된다


### RARP 에 대한 설명으로 타당하지 않은 것은 ?


: ARP 와 같은 역할을 수행하며 로컬디스크가 없는 시스템에 주로 사용된다


### 인터넷 그룹 관리 프로토콜로 사용자들로 하여금 멀티캐스트 그룹에 참여하고 멤버간의 관계를 유지할 수 있도록 하는 프로토콜은 ?


: IGMP


### 전자 메일 전송 프로토콜로 25번 포트를 사용하여 메시지를 전송하는 프로토콜은 ?


: SMTP


### DNS 에 대한 설명 중 적당하지 않은 것은 ?


: 정방향 조회는 IP 주소를 호스트 이름으로 변환해준다


### TCP/IP 의 디자인 목표로 타당하지 않은 것은 ?


: TCP 와 IP 의 완벽한 기능 분리


### 호스트의 IP 주소가 50.221.100.5 번지를 갖는다면 해당하는 네트워크의 Class 는 ?


: Class A


### IP Addressing 에 대한 설명으로 타당하지 않은 것은 ?


: Class B 의 최상위 2비트는 01 로 설정한다


### 서브넷 마스크에 대한 설명으로 적절하지 않은 것은 ?


: Class C 에서 다섯 개의 네트워크로 나누고자 한다면 실제 서브넷은 255.255.224.0 이 된다


### 패킷 전송의 최적 경로를 위해 다른 라우터들로부터 정보를 수집하는데 최대 홉이 15를 넘지 못하는 프로토콜은 ?


: RIP


### Link State 알고리즘을 이용해 서로에게 자신의 현재 상태를 알려주며 네트워크 내 통신을 위해 사용하는 프로토콜은 ?


: IGP


### Telnet 의 사용이 적절하지 않은 것은 ?


: Telnet 포트 이름


### 포트번호가 잘못 짝지어진 것은 ?


: Telnet - 24번


### rlogin 과 같은 초기 유닉스 계열 명령어에 보안 기능을 보완하여 원격지에 있는 호스트를 보다 안전하게 접속할 수 있는 프로토콜은 ?


: SSH


### NIC(Network Interface Card) 에 대한 설명으로 적합하지 않은 것은 ?


: OSI 7계층중 전송계층에서 사용된다


### TCP/IP 프로토콜 중 같은 계층에서 동작하지 않는 것은 ?


: SMTP


### 정보통신망 중에서 노드의 개수가 같다고 가정했을 경우 가장 많은 링크를 발생할 수 있는 것은 ?


: 메쉬망


### 컴퓨터 온라인 시스템에 사용하는 통신 제어 장치(CCU) 및 통신 제어 처리 장치(CCP)의 주요 기능이나 역할과 관계가 없는 것은 ?


: 아날로그와 디지털 신호의 변환


### 회선경쟁선택방식의 프로토콜에 관한 설명으로 옳은 것은 ?


: 트래픽이 많은 멀티포인트회선에 사용할 경우에는 비효율적이다


### 모뎀과 DTE(Data Terminal Equipment) 간에 이진 직렬 데이터 전송을 하기 위한 동작을 규정한 것은 ?


: V.24


### 데이터 프레임의 구성에 따른 프로토콜의 분류 방식이 아닌 것은 ?


: 페이지 방식


### 공중통신사업자로부터 통신회선을 빌려서 컴퓨터를 접속시킨 후 이 컴퓨터가 갖고있는 정보를 여러 사용자에게 재판매하는 통신 서비스 시스템은 ?


: VAN(Value Added Network)


### 패킷교환의 특징에 대한 설명 중 가장 적절하지 않은 것은 ?


: 복수의 상대방과는 통신이 불가능하다


### 메일 서비스와 가장 관계가 먼 프로토콜은 ?


: FTP


### 다이얼 접속에 사용되는 SLIP 과 PPP 프로토콜을 비교한 설명 중 가장 적절하지 않은 것은 ?


: SLIP 은 PPP 와 마찬가지로 암호화된 사용자 인증을 제공한다


### 패킷통신의 구조에 관한 내용 중 잘못된 것은 ?


: 회선 교환이 소프트웨어적인 개념의 교환이라면 패킷 교환은 하드웨어적인 개념의 교환이라고 볼 수 있다


### DNS 서버에 IP 주소를 도메인 네임으로 변환하는 기능을 가지도록 할 때 추가해야 하는 영역은 ?


: 역방향 조회 영역


### DNS 서버 중 주기적으로 보조 서버에 정보를 전송하는 서버는 ?


: 주 서버


### DNS 서버를 설치하기 위해 우선 제어판에서 프로그램 추가/제거 를 실행하는 이유는 ?


: Windows 2000 Server 의 기본 설치 시 자동으로 설치되지 않기 때문이다


### FTP사이트 등록정보의 FTP 사이트 탭에서 현재 FTP 사이트에 접속중인 사용자의 정보를 나타내는 방법 중 옳지 않은 것은 ?


: 10일 단위로 평균 사용자 수를 알 수 있다


### Windows 2000 Server 의 IIS 5.0 에서 지원되는 인증 방식이 아닌 것은 ?


: 복합인증


### 여러 종류의 파일 시스템을 한 개의 하드디스크에서 사용해야 할 경우 가장 적절한 것은 ?


: 하드디스크를 파티션한다


### IP 주소 할당 및 관리를 동적으로 설정할 수 있는 서버는 ?


: DHCP


### Active Directory 의 논리적 구조가 아닌 것은 ?


: Graph


### 기존 NT 4.0 까지 지원되던 도메인 모델이나 익스체인지 서버의 최신 버전에서 구현되었던 엔진을 모델로 하여 Windows 2000 에서 사용되는 서비스로 대규모 네트워크상에서 적합한 디렉터리 서비스는 ?


: Active Directory


### Windows 2000 Server 가 설치될 때 자동으로 생성되며 극히 제한적인 권한만 가지는 계정은 ?


: Guest


### 리눅스 시스템에서 지원하지 않는 파일 시스템은 ?


: EXT4


### 각 서비스에 대한 설명으로 옳은 것은 ?


: 고퍼(Gopher) - 인터넷 상에서 메뉴 방식의 자료 검색


### 사용자 프로파일에 대한 설명 중 잘못된 것은 ?


: 최초 로그온 시 서버로부터 이동 사용자 프로파일을 다운로드 받지 못하는 경우에는 로그온 할 수 없다.

: Deadlock


### 다음 중 LAN 구축 시 고려해야 할 요소가 아닌 것은 ?


: 전송성능


### 다음 중 VolP 주요기술에 해당되지 않는 것은 ?


: 지향성 안테나 기술


### Windows 2000 Server 에서 현재 공유 자원에 연결된 사용자들을 끊으려면 컴퓨터 관리의 MMC 트리에서 어떤 작업을 실행해야 하는가 ?


: 세션을 실행한다


### Windows 2000 Server 의 인터넷 정보 서비스 (IIS) 에서 사용 가능한 서비스는 ?


: FTP 서비스, 웹 서비스


### 다음 중 인터넷 정보 서비스(IIS) 메뉴에서 관리 웹 사이트 등록 정보에 없는 탭은 ?


: LDAP 라우팅


### 다음 중 Windows 2000 Server 에서 FTP Server 와 관련이 없는 것은 ?


: Exchange Server


### 다음 DNS  데이터 베이스 레코드의 유형 중 연결이 잘못된 것은 ?


: CNAME - 호스트 정보를 표시한다


### DHCP 서버는 Scope 이라 불리는 IP 주소의 범위를 만들어 컴퓨터들에게 제공할 IP 주소 정보를 준비한다 다음 중 Scope 의 속성으로 볼 수 없는 것은 ?


: DNS 서버 이름


### W 에서 전 네트워크에 있는 자원에 대한 정보를 논리적 계층 구조로 저장하고 있어서 이를 찾아내거나 관리하고 사용하기 용이하게 만든 것은 ?


: Active Directory 서비스


### W 의 사용자 관리에 관한 다음 설명 중 옳은 것은 ?


: 관리자가 만든 계정은 모두 삭제 가능하다


### 공유 폴더를 특정 사용자 또는 그룹만이 접근할 수 있도록 할 때 다음 중 옳은 것은 ?


: 현재 권한을 가지고 있는 모든 사용자를 삭제하고 원하는 사용자에게만 모든 권한을 부여한다


### W 의 사용자 계정에 대한 설명으로 옳은 것은 ?


: 계정을 삭제하지 않고도 사용하지 못하도록 할 수 있다


### 다음 중 파일과 폴더의 보안을 위한 NTFS 사용권한의 설명 중 맞는 것은 ?


: NTFS 는 파일 내에 보안 정보가 기록되기 때문에 파일과 폴더의 사용 권한을 각각 설정해 줄 수 있다


### W 의 계정에 대해 잘못 설명된 것은 ?


: 삭제된 로컬 사용자 계정은 같은 ID 로 등록하면 이전과 동일한 사용자로 인식한다


### Linux 의 LILO 에 대한 설명으로 잘못된 것은 ?


: LILO 는 하드 디스크의 MBR 에만 설치된다


### Linux 에서 파티션 할 경우 Type 이 Swap 인 경우는 어떤 의미인가 ?


: 메모리가 부족할 경우 하드디스크의 일부분을 마치 메모리인 것처럼 사용하는 파티션이다


### W 의 네트워크 방식 중에서 모든 계정과 자원을 특정 서버에서 관리하는 중앙 집중식 방식은 ?


: 도메인 방식


### Linux 의 기본 명령어들이 포함되어 있는 디렉터리는 ?


: /bin


### 도스의 명령어 해석기는 [command.com](http://command.com/) 이다 Linux 의 명령어 해석기는 ?


: Shel


### Linux 의 vi 명령어 중 문자 하나를 삭제할 때 사용하는 명령어는 ?


: x


### SCSI ID 설정방법과 거리가 먼 것은 ?


: CD-ROM, CD-R 등의 속도가 빠른 기기를 뒤에 설치함


### 미러링이라고 하며 최고의 성능과 최고의 고장대비 능력을 발휘하는 것은 ?


: RAID 1


### 라우터에 대한 설명 중 틀린 것은 ?


: OSI 7  Layer 중 Transport 층에 대응된다


### 다음 중 스위칭 허브 (Switching Hub) 에 관한 설명으로 맞는 것은 ?


: Port 당 속도가 일정하게 보장된다


### 서로 유사한 MAC(Medium Access Control) 프로토콜을 사용하는 LAN 사이를 연결하기 위해 사용되는 장비로 OSI 모델의 2계층에서 동작되는 것은 ?


: 브릿지(Bridge)
