`add(int index, E element)` : 특정 인덱스에 요소를 삽입


```java
list.add(1, "Banana"); // 1번 인덱스에 "Banana" 삽입
```


`remove(Object element)` : 특정 요소를 제거


```java
list.remove("Apple"); // "Apple" 제거
```


`get(int index)` : 특정 인덱스의 요소를 반환


```java
String item = list.get(0); // 0번 인덱스의 요소 반환
```


`indexOf(Object element)` : 특정 요소의 첫 번째 인덱스를 반환


```java
int index = list.indexOf("Apple"); // "Apple"의 인덱스 반환
```


`lastIndexOf(Object element)` : 특정 요소의 마지막 인덱스를 반환


```java
int index = list.lastIndexOf("Apple");
```


`size()` : 리스트의 요소 개수를 반환


```java
int size = list.size();
```


`isEmpty()` : 리스트가 비어 있는지 확인


```java
boolean empty = list.isEmpty();
```


`set(int index, E element)` : 특정 인덱스의 요소를 새로운 값으로 대체


```java
list.set(0, "Orange"); // 0번 인덱스를 "Orange"로 수정
```


`clear()` : 리스트의 모든 요소 제거


```java
list.clear();
```


`toArray()` : 리스트를 배열로 반환


```java
Object[] array = list.toArray();
```


`subList(int fromIndex, int toIndex)` : 리스트의 일부를 반환


```java
List<String> subList = list.subList(0, 2); // 0~1번 인덱스 요소 반환
```


`sort(Comparator<? super E> c)` : 리스트를 정렬


```java
list.sort(Comparator.naturalOrder()); // 오름차순 정렬
```


`forEach(Consumer<? super E> action)` : 리스트의 각 요소를 반복 처리


```java
list.forEach(item -> System.out.println(item));
```
