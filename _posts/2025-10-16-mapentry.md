맵의 단일 항목(entry)을 나타냄


**하나의 key-value 쌍을 저장하는 객체**로 이를 통해 Map의 항목에 순차적으로 접근하거나 수정할 수 있음


### 주요 메서드


`getKey()` : 항목의 키 반환


`getValue()` : 항목의 값 반환


`setValue(V value)` : 항목의 값 수정 (값 설정)


{% raw %}
```java
Map<String, String> map = new HashMap<>();
map.put("key1", "value1");

for (Map.Entry<String, String> entry : map.entrySet()) {
	if ("key1".equals(entry.getKey())) {
		entry.setValue("newValue1");
	} 
}

System.out.println(map);
```
