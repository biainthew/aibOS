# _data-_


: 요소에 특정한 데이터를 저장하고 싶은 경우 자유롭게 사용 가능


### ✔️ 문법


> 💡 `data-속성이름 = “속성값”`


👉🏻 하나의 요소에 여러 속성을 동시에 사용할 수 있고 코드가 간결해지는 장점이 있음


### 🌈 예시


> 💡 `<a data-num=”80”>`


👉🏻 임의로 만든 속성
