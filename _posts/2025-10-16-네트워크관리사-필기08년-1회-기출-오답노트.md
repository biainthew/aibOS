### 현재 사용하는 IP Address 체계에서 가장 많은 네트워크를 가지는 Class 는  ?


: C Class


### IP 는 데이터 세그먼트를 패킷으로 만들어 목적지로 전달하는 역할을 한다 이때 패킷을 MTU(Maximum Transmission Unit) 에 관한 설명으로 옳지 않은 것은 ?


: 패킷이 분할되었는지 그렇지 않은지 확인하는 필드는 Reserved Flag 필드이다


### Telnet 을 이용하여 상대방 컴퓨터에 접속을 시도하려고 한다 상대방 컴퓨터에게 일정한 데이터를 보내 상대방 컴퓨터의 정상 동작 여부를 확인하고 싶을 때 사용하는 명령어는 ?


: Ping


### DNS 에 대한 설명으로 옳지 않은 것은 ?


: 정방향 조회는 IP Address 를 호스트 이름으로 변환해 준다


### TCP/IP 망의 관리에 사용되는 프로토콜은 ?


: SNMP


### 가정이나 사무실에 이미 구축되어 있는 전력선을 이용하여 데이터를 전송하는 방법은 ?


: PLC


### CSMA/CD 의 특징으로 가장 옳지 않은 것은 ?


: 8B/5B 혹은 8B/10B 부호화 방식을 사용한다


### Ethernet 프레임의 전체 바이트 범위와 데이터 필드의 바이트 범위로 잘 짝지어진 것은 ?


: 64~1518, 46~1500


### IrDA 기술의 특징으로 옳지 않은 것은 ?


: 거리에 상관없이 사용이 가능하다


### IEEE 802.11 WLAN(무선랜) 접속을 위한 NIC 에서 사용하고 있는 다중접속 프로토콜은 ?


: CSMA/CA


### 저전력 저가격 사용의 편리성을 가진 근거리 무선네트워크의 대표적 기술 중의 하나로 IEEE 802.15.4 표준의 PHY 층과 MAC 층을 기반으로 상위 프로토콜과 응용을 규격화한 기술로 원격제어 및 관리의 응용에 적합한 홈 오토메이션 등에 적용되며 유비쿼터스 센서 네트워크 환경 구축에 중추적 역할을 담당할 기술은 ?


: ZigBee


### CATV 의 주요 구성 요소로 가장 옳지 않은 것은 ?


: 엑세스 포인트


### 원거리 통신의 전송방식 중 회선교환방식에 관한 설명은 ?


: 망 내 교환기들은 전송되고 있는 사용자 데이터에 대한 에러제어 기능을 수행하지 않고 단순히 데이터가 전송되는 경로만을 제공하며 송수신측은 동일한 속도로 동작하여야 한다


### VolP 에서 H.323 의 장점에 해당하는 것으로 옳지 않은 것은 ?


: PC 기반의 화상 회의 구축이 불가능


### 광섬유의 구조로 옳지 않은 것은 ?


: 석영 (Quartz)


### 직원이 300명인 회사에서 100명씩 3교대 근무하며 3교대 구성 그룹은 매주 50명씩 바뀐다 개인 데이터의 저장을 위해 노트북을 300명 모두에게 지급했으나 고정 IP 는 110 개 밖에 없다 이런 IP Address 부족을 해결하기 위하여 W 를 어떻게 구성하면 좋은가 ?


: DHCP 구성 (Dynamic host configuration protocol / 자동으로 IP 를 호스트하는 IP 주소 및 서브넷 마스크 및 기본 게이트웨이 등의 기타 관련된 구성 정보를 제공하는 클라이언트 / 서버 프로토콜)


### RDP(Remote Desktop Protocol) 에 대한 설명으로 옳지 않은 것은 ?


: RDP 를 이용한 터미널 서비스는 LAN (Local Area Network) 상에서만 연결을 지원한다


### W 의 구성요소를 캡쳐한 화면이다 FTP 와 관련 있는 구성 요소는 ?


: 인터넷 정보 서비스 (IIS)

### W 의 기능에서 독립적인 컴퓨터들을 그룹화 하여 공통의 응용 프로그램 집합을 실행하는 것으로 한 부분에서 에러가 발생하더라도 전체 시스템이 중단되지 않는 것은 ?


: Clustering


### W 에서 NTFS 파일 시스템에 대한 설명으로 옳지 않은 것은 ?


: NTFS 파일 시스템은 Windows 내의 프로그램으로 FAT 또는 FAT32 로 변환할 수 있다


### 하드웨어가 모두 정상이라는 가정 하에서 W 를 설치할 때 실패할 수 있는 원인으로 옳지 않은 것은 ?


: 파일 시스템을 FAT32 로 사용할 때


### W 의 기본 보안으로 옳지 않은 것은 ?


: User 및 Group 들의 검색 허용


### 명령 프롬프트에서 IIS 의 World Wide Web 서비스를 시작하기 위한 명령은 ?


: net start w3svc


### W 에서 사용되는 서비스로 대규모 네트워크 상에서 적합한 디렉터리 서비스는 ?


: Active Directory


### W 에서 프린터를 공유하여 같이 사용할 때 프린트를 요청했지만 프린트가 되지 않고 작업을 지울 수도 없을 때 문제를 해결하기 위해 이용하는 서비스는 ?


: Spooler Service


### W 에서 다음 DNS 데이터베이스 레코드의 유형 중 연결이 잘못된 것은 ?


: CNAME - 호스트 정보를 표시한다


### Linux 시스템 커널 버전에 대한 규칙으로 옳지 않은 것은 ?


: 버전의 두 번째 자리가 홀수이면 안정 버전 짝수이면 개발 버전을 뜻한다


### Linux 설치 과정에서 방화벽 설정을 최상위 수준으로 선택하였을 때 차단되지 않는 서비스는 ?


: TCP/IP 서비스


### LAN Card 의 IP Address 를 세팅할 때 디폴트 게이트웨이 주소는 일반적으로 어디 주소를 의미하는가 ?


: 해당 PC 가 접속되어 있는 라우터의 IP Address


### Cisco 2501 Router 의 Port 구성 중 내부 LAN 을 연결하기 위한 인터페이스는 ?


: AUI Ethernet Port


### 케이블의 두께가 사람의 머리카락 굵기만큼 가는 유리섬유로 정보를 보내고 받는 속도가 가장 빠르고 넓은 대역폭을 갖는 것은 ?


: Optical Fiber


### IP Address 의 설명 중 가장 옳지 않은 것은 ?


: 네트워크 구획상의 각 호스트는 같은 호스트 IP Address 를 갖는다


### TCP/IP 에 대한 설명으로 가장 올바른 것은 ?


: 호스트에 할당되는 IP Address 방식은 네트워크의 규모에 따라 일반적으로 ABC 세 개의 Class 로 구성된다


### Anonymous FTP 에 대한 설명으로 옳지 않은 것은 ?


: 일반적으로 80번 포트를 사용하여 상호 통신한다


### 공인 IP Address 와 사설 IP Address 를 매핑하는 기술은 ?


: NAT


### 인터넷 메일 호스트 사이에 아스키형식 이외의 텍스트 및 화상이나 음성 영상 등의 멀티미디어 데이터를 아스키 형식으로 변환할 필요 없이 인터넷 전자우편으로 송신하기 위한 인터넷 표준은 ?


: MIME


### 원격에 있는 호스트 접속 시 암호화된 패스워드를 이용하여 보다 안전하게 접속할 수 있도록 rlogin 과 같은 프로토콜을 보완하여 만든 프로토콜은 ?


: SSH


### 인터넷을 경유하여 로컬 네트워크에 접근할 때 보안을 강화하기 위해 사용하는 프로토콜은 ?


: PPTP


### HomePNA 기술에서 데이터 전송에 사용하는 표준 전송 방식은 ?


: CSMA/CD


### SNMP 에 대한 설명으로 옳지 않은 것은 ?


: TCP 를 이용하여 신뢰성 있는 통신을 한다


### LAN 의 특징으로 가장 옳지 않은 것은 ?


: 라우팅과 같은 경로 선택이 필요하다


### 최대 전송속도는 4Mbps 이고 전송될 수 있는 가장 긴 데이터 길이는 2048Byte 이며 차폐물이 가로막고 있을 경우에는 통신이 불가능한 무선 근거리 통신기술은 ?


: IrDA


### 대부분의 무선기술들이 무선 주파수 반송파라 불리는 기준 주파수 파형의 형태를 변화시켜 정보를 전달하는데 반해 이 기술을 반송파를 사용하지 않고 0과 1 처럼 일정한 주기와 파형을 가지고 있는 전기적 신호인 펄스를 1나노초 보다 짧은 시간 간격으로 안테나를 통해 전송하는 통신방식


: UWB (Ultra-wideband, 초광대역


### ADSL 에 대한 내용으로 잘못된 것은 ?


: 전화국과 연결하기 위해 일반 랜 카드가 필요하며 랜카드와 전화선을 연결하기 위한 Splitter 라는 기계가 필요하다


### - A 사의 네트워크는 규모 확장으로 IP Address 가 부족한 상태가 발생하였다, 현재 네트워크는 전용선으로 묶여있으며 사용가능한 공인 IP 는 64개이다, 네트워크에 연결되어야 할 PC 는 총 120 대이고 모든 컴퓨터는 외부에서의 접속은 필요 없다, 가장 적은 비용으로 동시에 120대가 모두 인터넷에 접속할 수 있도록 하고자 한다


: Router 의 NAT 기능을 활용한다


### 광파이버 (Optic Fiber) 의 굴절률이 전파하는 광의 파장에 의해 변화함으로써 생기는 파형의 퍼짐을 뜻하는 것은 ?


: 재료 분산


### Linux 시스템에 설치된 입출력 장치들과 마운트 될 파일 시스템의 마운트 포인터가 위치하는 디렉터리는 ?


: /mnt


### W 에서 DHCP 를 사용해 IP Address 가 동적으로 할당되도록 하였다 필요에 의해 특정 클라이언트가 특정 IP Address 를 할당받게 하기 위해서는 아래의 어느 메뉴 항목에서 설정해야 하는가 ?


: 예약


### W 에서 그룹 계정을 사용하는 이유는 그룹에 권한을 설정하여 사용자 계정을 포함시키면 관리하기 편리해지기 때문이다 그룹 계정의 종류에 해당하지 않는 것은 ?


: Network Group


### W 가 감시할 수 있는 이벤트의 종류를 설명한 것 중 잘못된 것은 ?


: 디렉터리 서비스 엑세스 - 사용자가 파일, 폴더 또는 프린터에 엑세스 하는 이벤트


### W 의 보안 구성 작업 자동화에 관한 설명으로 옳지 않은 것은 ?


: 템플릿이 미리 작성되어 있어야 이용 가능하다


### Linux 의 kill 명령어에 대한 설명이다 가장 옳지 않은 것은 ?


: kill_프로세스_번호 를 사용하면 언제나 원하는 프로세스를 종료시킬 수 있다


### W 의 시스템 성능 모니터 의 내용으로 옳지 않은 것은 ?


: 시스템의 정확한 작동 정보를 위한 베이스 라인의 설정은 다수의 사용자가 세스템을 이용할 때 설정한다


### W 에서 원격으로 서버에 접속하여 GUI 환경으로 관리자 기능을 사용할 수 있는 서비스는 ?


: 터미널 서비스


### 보조 DNS 에 대한 설명으로 가장 옳지 않은 것은?


: 도메인 데이터베이스 수정은 보조 DNS 서버와 주 DNS 서버 동시에 이루어진다


### W 에서 구성된 FTP 사이트를 변경할 때 FTP 사이트 탭과 관련 없는 것은 ?


: 익명 연결 허용


### W 의 이벤트 뷰어에서 제공되는 이벤트 유형에 관한 설명으로 옳지 않은 것은 ?


: 정보 - 서비스 수행에서 필요한 데이터 정보를 기록한다


### W 에서 IIS 5.0 을 사용하여 구축하는 서버로 옳지 않은 것은 ?


: Terminal Server


### W 의 디폴트 보안 템플릿에 관한 설명으로 옳지 않은 것은 ?


: W 설치 플랫폼 에 관계없이 동일하게 적용된다


### 광섬유에서 광의 입사각 조건을 표시하는 것은 ?


: 개구수 (NA)


### 버스형 또는 데이지 체인형 네트워크의 종단에 부착하는 장치로 신호를 흡수함으로써 다시 반향되지 않도록 하는 장비는 ?


: Terminator


### 무선 랜의 구성 방식 중 무선랜 카드를 가진 컴퓨터 간의 네트워크를 구성하여 작동하는 방식은 ?


: Ad Hoc
