### 개요

1. 인터넷에서 가장 많이 사용되는 표준 프로토콜
2. OSI 계층의 전송계층과 네트워크 계층에서 동작하고 상위의 응용프로그램을 구별한다

### TCP

1. 데이터의 전송 방법을 결정하는 프로토콜
2. OSI 7 계층 중 전송계층에 해당하며 신뢰성있는 연결형 서비스를 제공
3. 패킷의 다중화, 순서제어, 오류제어, 흐름제어 기능을 제공
4. 강력한 에러 제어 기능이 있어서 정확한 데이터 전송
5. 슬라이딩 윈도우 방식
6. 패킷의 순차번호와 CRC 이용하여 신뢰성 있는 통신 수행
7. 프로토콜 : FTP, TELNET, HTTP, SMTP, POP

### IP

1. 인터넷상에서 각 호스트의 주소를 결정하는 프로토콜
2. OSI 7 계층의 네트워크 계층에 해당하며 데이터그램 기반으로 하는 비연결형 서비스
3. 패킷의 분해/조립, 주소지정, 경로선택 기능을 제공

### TCP/IP 계층과 각 프로토콜

1. 네트워크 계층(물리계층, 데이터링크계층) : Network Interface
2. 인터넷 계층(네트워크계층) : IP, ARP, RARP, ICMP
3. 전송계층 : TCP, UDP
4. 응용계층(세션, 표현, 응용계층) : FTP, TELNET, TFTP, DNS, SMTP, SNMP
