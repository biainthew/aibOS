### NIC(Network Interface Card) 에 대한 설명으로 옳지 않은 것은 ?


: OSI 7 Layer 중 전송(Transfer) 계층에서 사용된다


### 다음 중 TCP/IP 응용계층 프로토콜로 옳지 않은 것은 ?


: ARP


### 호스트의 IP Address ‘50.221.100.5’ 에 해당하는 네트워크의 Class 는 ?


: A Class


### 호스트 어드레스와 네트워크 어드레스를 구분하게 하는 것으로 어드레스 제어를 쉽게 할 수 있게 하는 방법은 ?


: Subnet Mask


### IP Header 의 내용 중 TTL(Time To Live) 의 기능을 설명한 것 중 옳지 않은 것은 ?


: IP 패킷은 네트워크 상에서 영원히 존재할 수 있다


### OSI 7 Layer 의 전송계층에 속하며 사용자에게 신뢰할 수 있는 데이터 전송을 지원하는 인터넷 프로토콜은 ?


: TCP


### UDP 헤더 구조에 대한 설명으로 가장 옳지 않은 것은 ?


: Destination Port - 선택적 필드로 사용하지 않을 때는 zero 로 채워지는 필드


### ARP 와 RARP 에 대한 설명으로 옳지 않은 것은 ?


: RARP 는 브로드 캐스팅을 통해 해당 네트워크 주소에 대응하는 하드웨어의 물리적 주소를 얻는다


### ICMP 의 Message Type 필드의 유형과 질의 메시지 내용이 잘못 연결된 것은 ?


: 5 - Echo Request - 네트워크상에서 두 개 이상 장비의 기본 연결을 검사하기 위해 사용된다


### TCP/IP 프로토콜 중에서 IP 계층의 한 부분으로 에러 메시지와 같은 상태 정보를 알려주는 프로토콜은 ?


: ICMP(Internet Control Message Protocol)


### 다음 중 NetBIOS 이름을 IP Address 로 사상(Alias) 하기 위해 사용되는 것은 ?


: WINS


### Telnet 접속 방법으로 옳지 않은 것은 ?


: Telnet 포트 번호


### URL에 대한 설명 중 옳지 않은 것은 ?


: URL 은 지정된 컴퓨터를 식별하기 위해 사용되는 것으로서 전 세계적으로 유일한 이름을 가질 뿐만 아니라 하나의 도메인 이름은 오직 하나의 호스트와 대응할 수 있다


### 다음 프로토콜 중에서 하이퍼 미디어 하이퍼텍스트를 통한 클라이언트와 서버간의 통신에 이용되는 것은 ?


: HTTP


### FTP 에 관한 설명으로 가장 옳은 것은 ?


: 서로 다른 컴퓨터 사이의 파일 전송서비스를 제공한다


### 인터넷 메일 확장자로서 사용자간의 바이너리 파일전송에 관한 인터넷 표준안을 지칭하는 것은 ?


: MIME


### TCP/IP 계층 모델 중 인터넷 계층에 속하는 것으로 멀티캐스트 그룹멤버 관리와 관련된 역할을 수행하는 프로토콜은 ?


: IGMP


### 디지털 전송의 특징으로 옳지 않은 것은 ?


: 신호 재생성기로 증폭기(Amplifier) 를 이용한다


### 문자열을 블록으로 전송하고 타이밍 신호는 터미널 모뎀이 공급하고 블록 내에서는 휴지 간격이 없고 중 고속 전송에 적합한 전송방식은 ?


: 동기 전송방식


### 일반적으로 PC 에서 TCP/IP 망을 구성하는데 필요한 항목으로 가장 옳지 않은 것은 ?


: IPX/SPX 주소


### 메일 서비스와 가장 관계가 먼 프로토콜은 ?


: FTP


### 네트워크의 관리 및 네트워크의 장치와 그들의 동작을 감시, 관리하는 프로토콜은 ?


: SNMP


### 다음의 망 연동 장치 중 네트워크 계층에서 사용하며 여러 개의 서브 네트워크를 연결할 때 사용하는 것은 ?


: Router


### 송신측에서 여러 개의 터미널이 하나의 통신 회선을 통하여 신호를 전송하고 전송된 신호를 수신측에서 다시 여러 개의 신호로 분리하는 것은 ?


: 다중화장치(Multiplexing)


### 전송계층에서 동작하는 프로토콜만으로 구성된 것은 ?


: TCP, UDP


### IEEE 표준안 중 Ethernet 에 관한 표준안은 ?


: IEEE 802.3


### 통신을 원하는 두 실체 (Entity) 간에 무엇을 어떻게 언제 통신할 것인가에 대해 서로 약속한 일련의 절차나 규범의 집합은 ?


: Protocol


### 네트워크 인터페이스가 내장된 프린터 디바이스의 사용으로 관계가 없는 것은 ?


: 네트워크 인터페이스 프린터 디바이스 포트 이름은 URL 방식으로 입력한다


### IIS(인터넷 정보 서비스) 에서 작업할 수 있는 서비스로 옳지 않은 것은 ?


: DHCP(Dynamic Host Configuration Protocol)


### IIS 웹 서버의 설정 중 하루에 평균적으로 방문하는 사용자 수를 설정하기 위해 사용하는 탭은 ?


: 성능 탭


### 다음 중 DNS 서버 종류로 옳지 않은 것은 ?


: 슬레이브 네임 (Slave name Server)


### FTP 서버 운영자가 본인이 관리하는 FTP 서버의 세션 관리 중에서 할 수 없는 것은 ?


: 다운로드 할 수 있는 파일의 형식 제한


### 네트워크 관리자인 당신은 보안상의 이유로 FTP 서버의 기본 포트를 변경하려고 한다 당신의 네트워크 서버에는 메일서버와 함께 웹 서버 FTP 서버도 같이 있다 다음 중 사용 가능한 포트 번호는 ?


: 81


### W 의 WINS 서버에 대한 설명으로 옳지 않은 것은 ?


: WINS 서버가 실행되는 호스트는 동적 IP Address 를 사용하여야 한다


### DHCP 서버에 대한 설명으로 맞는것은 ?


: 특정 시점의 네트워크에 연결된 컴퓨터에 동적으로 IP Address 를 할당한다


### Active Directory 의 논리적 구조가 아닌 것은 ?


: Graph


### W 의 설치 시에 기본적으로 설치되는 것은 ?


: TCP/IP


### 시스템 인터페이스 정보를 출력하면서 송수신 패킷 수 및 오류수 충돌횟수 현재 출력 큐에 대기중인 패킷 등에 관한 정보를 제공하는 것은 ?


: Netstat


### 파티션을 FAT 에서 NTFS 로 변환하는 명령은 ?


: convert C: /FS:NTFS


### 한 개의 시스템 C 드라이브에 W 와 Windows 98 로 멀티부팅을 사용하기 위한 시스템 설정으로 가장 올바른 것은 ?


: 하드디스크의 주 파티션을 FAT 로 포맷한다


### W 의 계정에 대한 설명으로 잘못된 것은 ?


: 삭제된 로컬 사용자 계정은 같은 ID 로 등록하면 이전과 동일한 사용자로 인식한다


### 사용자의 명령어를 Kernel 에 전달해주는 역할을 하는 것은 ?


: Shell


### 다른 운영체제와 Linux 가 공존하는 하나의 시스템에서 멀티 부팅을 지원할 때 사용되며 Linux 로더를 의미하는 것은 ?


: LILO


### 특정한 파일을 찾고자 할 때 사용하는 Linux 명령어는 ?


: find


### Linux 에서  /home 디렉터리 밑에 icqa 라는 하위 디렉터리를 생성하고자 할 때 올바르게 사용된 명령어는 ?


: mkdir /home/icqa


### 루핑 현상을 방지할 수 있는 네트워크 경로 설정 방법은 ?


: 스패닝 트리 기능의 사용


### 네트워크 트래픽을 해소하기 위한 기법의 하나로써 허브와 허브를 연결하는 방식 가운데 대역폭을 넓힐 수 있는 연결방식은 ?


: Port Trunk


### 버스형 또는 데이지 체인형 네트워크의 종단에 부착하는 장치로서 신호를 흡수합으로써 다시 반향되지 않도록 하는 것은 ?


: terminator


### 광섬유의 특성 중 옳지 않은 것은 ?


: 손실률이 높다


### 2.4GHz에서 서비스를 제공하며 초당 11Mbps 의 속도를 표준으로 하는 무선 랜 규격은 ?


: 802.11b


### 패킷 전송의 최적 경로를 위해 다른 라우터들로부터 정보를 수집하여 라우팅 테이블에 저장하게 된다 이때 사용되지 않는 프로토콜은 ?


: SMTP


### 서브넷 마스크에 대한 설명으로 옳지 않은 것은 ?


: 서브넷 마스크는 Network ID 필드는 0으로 Host ID 필드는 1로 채운다


### TCP/IP 인터네트워킹 구조에서 라우터가 포함할 수 있는 계층으로 옳지 않은 것은 ?


: 응용계층


### IP Address 에 대한 설명 중 옳지 않은 것은 ?


: IP Address 는 40 bit 로 구성되어 있다


### TCP 를 직접 이용하는 응용 프로그램으로만 묶인 것은 ?


: Telnet, SMTP


### 목적지 시스템의 32bit IP Address에 대응되는 48bit 의 하드웨어 네트워크 주소를 제공하는 프로토콜은 ?


: ARP


### TCP/IP 프로토콜 중 같은 계층에서 동작하지 않는 것은 ?


: SMTP


### 단말기 사용자나 클라이언트가 네트워크를 통해 원격 접속하여 마치 바로 옆에 있는 컴퓨터처럼 자유롭게 사용할 수 있도록 해 주는 프로토콜은 ?


: Telnet


### TCP/IP 의 응용계층 프로토콜에 해당하는 것은 ?


: 웹 검색을 가능하게 하는 HTTP(Hyper Text Transfer Protocol)


### 인터넷의 도메인네임의 설명 중 옳지 않은 것은 ?


: 도메인 네임에 언더바나 대시 기호를 사용할 수 있다


### 도메인 네임을 아이피 어드레스로 변환하는 것은 ?


: DNS


### 컴퓨터 온라인 시스템에 사용하는 통신 제어 장치 및 통신 제어 처리 장치의 주요 기능이나 역할과 관계가 없는 것은 ?


: 아날로그와 디지털 신호의 변환


### 인터페이스의 주요 특성으로 옳지 않은 것은 ?


: 화학적 특성


### 병렬통신(Parallel Communication) 과 비교하여 직렬통신(Serial Communication) 의 특징으로 옳지 않은 것은 ?


: 컴퓨터와 주변기기 사이의 데이터 전송에 사용된다


### 공중 통신 사업자로부터 통신 회선을 임대하여 컴퓨터를 접속시킨 후 이 컴퓨터가 갖고 있는 정보를 여러 사용자에게 재판매 하는 통신 서비스 시스템은 ?


: VAN(Value Added Network)


### 다중화의 방법으로 옳지 않은 것은 ?


: TDMA


### IEEE 802 시리즈에 관한 내용이다 연결이 올바른 것은 ?


: 802.2 - LLC(Logical Link Control)


### 근거리 통신망(LAN) 에서 사용하는 CSMA/CD 프로토콜에 대한 설명으로 옳지 않은 것은 ?


: 공장용 프로토콜로 MAP(Manufacturing Automation Protocol) 에 주로 사용된다


### OSI 7 Layer 중 물리적 링크간의 신뢰성 있는 정보전송을 제공하며 동기화 에러제어 흐름제어로서 데이터의 블록을 전송하는 기능을 가진 계층은 ?


: 데이터 링크 계층


### 프로토콜을 형성하는 기능 중 각 계층의 데이터에 제어 정보를 부착하는 것을 뜻하는 것은 ?


: 캡슐화


### 하나의 웹 사이트에 Directory 별로 인증과 권한을 다르게 설정할 경우 일반적으로 사용하는 Directory 는 ?


: Virtual Directory


### IIS 웹 서버의 기능이 아닌 것은 ?


: NEWS 그룹 기능


### 도메인 이름의 구성 체계요소로 옳지 않은 것은 ?


: 사용자 ID


### W 에서 새로운 사용자를 만들어 줄 때 사용자 암호를 사용자만이 알게 하고 싶을 경우 관리자가 설정해야 할 옵션으로 가장 올바른 것은 ?


: 다음 로그온 할 때 반드시 암호 변경


### 현재 FTP 사이트에 접속 중인 사용자를 확인할 경우 FTP 서버의 등록정보에서 어느 탭을 사용하는가 ?


: FTP 사이트


### DHCP 서버 서비스를 설치한 후 주소 범위를 설정하려고 한다 이 때 주소 범위 설정에 사용되지 않는 옵션은 ?


: DHCP 서버의 MAC 주소


### W 에서 전 네트워크에 있는 자원에 대한 정보를 논리적 계층 구조로 저장하고 있어서 이를 찾아내거나 관리하고 사용하기 용이하게 만든 것은 ?


: Active Directory 서비스


### 한정된 IP Address 를 효율적으로 활용하기 위하여 W 에서 정해진 범위 내의 IP Address 를 자동적으로 할당해 주는 서버는 ?


: DHCP


### SQL 서버 7.0 의 클라이언트 소프트웨어를 지원하지 않는 운영체제는 ?


: Windows 3.1


### W 와 데이터센터에 있는 기능으로 한 컴퓨터에서 결함이나 오류가 발생할 경우 클러스터의 다른 컴퓨터를 이용해서 같은 서비스를 제공할 수 있는 기능은 ?


: 클러스터 서비스


### Active Directory 의 장점이 아닌 것은 ?


: 낮은 시스템 요구사항


### 그룹 파일이나 폴더의 소유권을 수행할 수 있는 영역은 ?


: Administrators


### W 의 Active Directory 서비스를 효과적으로 설치하기 위해서 고려해야 할 조건으로 가장 올바른 것은 ?


: Active Directory 서비스가 설치될 파티션은 NTFS 파일 시스템으로 포맷되어 있어야 한다

: 공유 폴더는 생성과 동시에 모든 사용자에게 자동적으로 접근이 허가된다

: DOS

### LAN 을 계층적으로 구성할 때 일반적으로 사용되는 장비로서 현재 저렴한 가격으로 가장 많이 사용되는 장비는 ?


: Hub


### 세션계층, 프리젠테이션 계층, 응용계층 간을 연결하여 메시지 형식변환, 주소변환, 프로토콜변환 기능을 갖는 장비는 ?


: Gateway


### 위성 통신이 다른 통신에 비해 우수한 점으로 옳지 않은 것은 ?


: 시간 지연이 없다
