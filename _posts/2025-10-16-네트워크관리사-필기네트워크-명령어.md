### ipconfig


: 자신의 IP 주소 나타내줌


### arp


: 데이터를 교환하는 컴퓨터의 IP 주소와 물리적인 주소 목록을 잠시 보관하는 arp cache 를 볼 수 있도록 출력


### netstat


: 네트워크를 통한 접속종류와 상태를 보여줌


### nslookup


: IP 로 도메인이름을 알아보거나 도메인 이름으로 IP 를 확인할 때 사용


### ping


: 네트워크가 연결이 잘 안 될 때 임의의 주소로 핑을 날려서 잘 받았나 확인해보는 명령어


### tracert


: 네트워크의 병목현상 확인
