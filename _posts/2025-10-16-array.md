### .at()


**: 배열에서 인덱스로 요소 찾기**


```javascript
const colors = ['빨강', '초록', '파랑'];

console.log(colors.at(2)) //'파랑'
```


### .concat()


**: 주어진 배열이나 값들을 기존 배열에 합쳐서 새 배열을 반환**


: 기존 배열을 변형하지 않음


: 추가된 새로운 배열을 반환


### .copyWithin()


**: 배열의 일부를 얕게 복사한 뒤 동일한 배열의 다른 위치에 덮어쓰고 그 배열을 반환**


: 배열의 길이를 수정하지 않고 반환


```javascript
arr.copyWithin(target, start, end)
```


`target` : 복사한 값을 넣을 위치를 가리키는 0 기반 인덱스


`start` : 복사를 시작할 위치를 가리키는 0 기반 인덱스


`end` : 복사를 끝낼 위치를 가리키는 0 기반 인덱스 / end 인덱스 이전까지 복사하므로 end 인덱스가 가리키는 요소는 제외


```javascript
[1, 2, 3, 4, 5].copyWithin(-2);
//[1, 2, 3, 1, 2]

[1, 2, 3, 4, 5].copyWithin(0, 3);
// [4, 5, 3, 4, 5]

[1, 2, 3, 4, 5].copyWithin(0, 3, 4);
// [4, 2, 3, 4, 5]

[1, 2, 3, 4, 5].copyWithin(-2, -3, -1);
// [1, 2, 3, 3, 4]
```


### .every()


**: 배열 안의 모든 요소가 주어진 판별 함수를 통과하는지 테스트**


```javascript
function isBigEnough(element, index, array){
	return element >= 10;
}
[12, 5, 8, 130, 44].every(isBigEnough); //false
[12, 54, 18, 130, 44].every(isBigEnough); //true
```


### .fill()


**: 배열의 시작 인덱스부터 끝 인덱스의 이전까지 정적인 값 하나로 채움**


```javascript
arr.fill(value, start, end)
```


`value` : 배열을 채울 값


`start` : 시작 인덱스, 기본값 0


`end` : 끝 인덱스


### .filter()


**: 주어진 함수의 테스트를 통과하는 모든 요소들을 모아 새로운 배열로 반환**


```javascript
arr.filter(callback(element, index, array))
```
