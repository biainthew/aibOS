### 다음 중 TCP/IP 프로토콜 계층구조가 하위층에서 상위층으로 바르게 배열된 것은 ?


: 네트워크 접속계층 - 인터넷 계층 - 전송계층 - 운영계층


### IP Address 체계에서 가장 많은 네트워크를 수용할 수 있는 클래스는 ?


: C Class


### 인터넷의 잘 알려진 포트번호와 맞지 않는 것은 ?


: TFTP - 60


### 서브 네트워킹의 장점이라고 할 수 없는 것은 ?


: IP Address 의 네트워크 ID 를 이용하므로 호스트 수가 확대되는 효과를 얻을 수 있다


### TCP/IP 망의 관리에 사용되는 프로토콜은 ?


: SNMP


### 소규모 네트워크에 적합하며 라우팅을 지원하지 않아 대규모의 네트워크나 WAN 에는 적합하지 않은 프로토콜은 ?


: NetBEUL


### 인터넷의 잘 알려진 포트번호로 맞지 않는 것은 ?


: TFTP - 68


### 서브 네트워킹의 장점이라고 할 수 없는 것은 ?


: IP Address 의 네트워크 ID 를 이용하므로 호스트 수가 확대되는 효과를 얻을 수 있다


### TCP/IP 망의 관리에 사용되는 프로토콜은 ?


: SNMP


### 소규모 네트워크에 적합하며 라우팅을 지원하지 않아 대규모의 네트워크나 WAN 에는 적합하지 않은 프로토콜은 ?


: NetBEUI


### UDP 에 대한 설명 중 올바른 것은 ?


: 비 연결성 데이터그램 서비스를 제공한다


### IP Address 를 하드웨어 주소로 매칭하는데 사용되는 것은 ?


: ARP


### SNMP 에 대한 설명 중 잘못된 것은 ?


: 보안 기능이 강화


### 원격지의 호스트에 연결하여 접속하려 할 때 이용할 수 있는 서비스는 ?


: Telnet


### DNS 레코드에 대한 설명으로 잘못된 것은 ?


: AAAA - A 레코드와 같으나 처리 속도를 증가시켰다


### 4KHz 의 음성신호를 PCM 전송을 할 때 사용하는 샘플링 주파수는 ?


: 8KHz


### 4위상 변조를 갖는 변조기가 2400baud 의 변이가 일어날 때 속도는 ?


: 4800bps


### 데이터 전송 시 전송매체를 통한 신호의 전달속도가 주파수의 가변적 속도에 따라 왜곡되는 현상은 ?


: 지연왜곡 (Delay Distortion)


### 패킷 교환망의 특징이 아닌 것은 ?


: 망에 유입되는 데이터의 양이 많아질 수록 전송속도가 빠르다


### 다음은 IEEE 802 규약이다 잘못짝지어진 것은 ?


: IEEE 802.2 - LDAP


### 다음 중 VDSL 에 대한 설명 중 잘못된 것은 ?


: 상향속도는 1.6 ~ 640Kbps 정도임


### 광섬유 케이블의 이점이 아닌 것은 ?


: 설치 및 유지보수가 쉽다


### 다음 중 동적인 불완전성의 원인이 아닌 것은 ?


: 하모닉 왜곡 - 신호의 감쇄가 신호의 진폭에 따라 달라지는 것을 말함


### HDLC 프레임의 구성에서 제어필드의 용도를 설정하는 것과 맞지 않은 것은 ?


: 비용형식 (C : Cost Pattern)


### W 의 DNS 서비스 중에서 해당 서버의 팝업메뉴가 아닌 것은 ?


: 새 로컬


### W 의 컴퓨터 관리 - 시스템 도구 - 공유 폴더 의 하위 항목이 아닌 것은 ?


: 로컬 파일


### 다음 서버와 관련 있는 것은 ?
Primary Name Server / Secondary Name Server / Master Name Server / Cashing Only Server


: DNS Server


### 다음 중 W 에서 FTP Server 와 관련이 없는 것은 ?


: Exchange Server


### W 의 프로그램 추가/제거 - Window 구성요소 추가/제거 메뉴를 선택한 것을 캡쳐한 화면이다 FTP 와 관련이 있는 것은 ?


: 인터넷 정보 서비스(IIS)


### 다음 FTP 명령어 중 원격지 호스트의 작업 디렉터리에서 a 로 시작하는 모든 파일을 지역 호스트의 작업 디렉터리에 복사하는 명령어는 ?


: ftp> mget a*


### DNS 데이터베이스 레코드 유형 중 호스트 이름을 주소로 매핑하는 것은 ?


: A


### 자동으로 호스트의 네트워크 환경을 설정할 수 있는 W 서비스는 ?


: DHCP (Dynamic Host Configuration Protocol)


### W 에서 인트라넷 상의 네트워크 접근을 위한 폴더 공유에 관한 설명으로 잘못된 것은 ?


: 공유 폴더에 동시 연결 접근자 수는 Server 의 경우 연결 가능한 지정 라이센스만큼 접근할 수 있다


### 사용자 계정이나 그룹을 이용하여 폴더에 NTFS 사용권한을 부여하는 경우 기본 NTFS 사용 권한이 아닌 것은 ?


:암호 지정


### Linux 시스템에서 현재 사용 디렉터리 위치에 상관없이 자신의 HOME Directory 로 이동하는 명령은 ?


: cd ~


### DHCP 관리자는 특정 IP 주소를 특정 호스트가 예약하여 사용하는 것을 허용한다 예약 설정을 하기 위해서 필요한 항목은 ?


: 해당 호스트의 MAC Address


### 다음 중 W 의 계정에 대한 설명이 틀린 것은 ?


: Guest 계정은 삭제가 가능하다


### Netstat 명령이 제공하는 정보가 아닌 것은 ?


: IP 패킷이 목적지에 도착하기 위해 방문하는 게이트웨이의 순서 정보


### Linux 에서 부팅디스켓을 만들 경우 사용하는 명령어는 ?


: dd


### W 에서 TCP/IP 가 정확히 설치되었는지 확인하는 방법으로 올바르지 않은 것은 ?


: ipconfig / renew


### W 에서 원격 설치 서비스를 구성하기 위하여 원격설치서버상에 구현이 되어 있어야 하는 네트워크 서비스가 아닌 것은 ?


: SMTP 서비스


### Linux 에 대한 일반적인 설명으로 틀린 것은 ?


: 서버용으로만 개발된 운영체제이다


### 루핑 현상을 방지할 수 있는 네트워크 경로 설정 방법은 ?


: 스패닝 트리 (Spanning Tree) 기능의 사용


### 네트워크 트래픽을 해소하기 위한 기법의 하나로써 허브와 허브를 연결하는 방식 가운데 대역폭을 넓힐 수 있는 연결방식은 ?


: Port Trunk


### Bridge 의 단점을 보완하고 경로 배정의 기능이 있는 네트워크 계층과 관련 있는 인터넷 연동 장치는 ?


: Router


### 다음 중 광섬유 (Fiber Optics) 의 설명과 거리가 먼 것은 ?


: 케이블의 크기가 상대적으로 작고 가볍기 때문에 설치가 쉽다


### 2.4GHz 에서 서비스를 제공하며 초당 11Mbps 의 속도를 표준으로 하는 무선 랜 규격은 ?


: 802.11b


### IP 의 체크섬(Checksum) 에 대한 설명으로 올바른 것은 ?


: IP 헤더의 완전성을 검사한다


### 다음 중 APP Cache 가 갱신되는 경우로 타당하지 않은 것은 ?


: RARP Request 를 보낸 경우


### 다음 중 서브넷 마스크에 대한 설명으로 옳지 않은 것은 ?


: 서브넷 마스크는 클래스 ABC 의 IP Address 는 항상 같은 서브넷 마스크를 사용한다


### 라우팅 테이블에 포함되어있는 정보로 옳지 않은 것은 ?


: 송신지 IP Address


### RIP 프로토콜의 일반적인 특징을 기술한 것으로 옳지 않은 것은 ?


: 멀티캐스팅을 지원한다


### 다음 중 도메인 네임으로 IP Address 를 조회할 수 있는 명령은 ?


: nslookup


### TCP/IP 응용계층에서 지원하지 않는 프로토콜은 ?


: IGMP(Internet Group Management Protocol)


### 다음 중 ARP Cache 를 지우는 명령어는 ?


: arp -d


### ICMP 의 Message Type 필드 번호와 유형이 잘못 연결된 것은 ?


: 12 - Timestamp Request


### 호스트 컴퓨터 aaa.bbb.ccc에 텔넷으로 접속하기 위해 telnet aaa.bbb.ccc : 9999 라고 명령을 입력한 경우 9999가 뜻하는 것은 ?


: 포트 번호


### 인터넷의 WWW에서 하이퍼 미디어, 하이퍼 텍스트를 통한 클라이언트와 서버간의 통신에 이용되는 TCP/IP 응용 프로토콜은 ?


: HTTP


### 원격 파일전송 서비스에서 등록되어 있지 않은 익명의 일반 사용자를 위한 계정은 ?


: Anonymous


### SMTP 에 대한 설명 중 가장 옳지 않은 것은 ?


: 110번 포트를 이용한다


### 다음 중 SNMP 의 설명으로 옳지 않은 것은 ?


: SNMP 는 TCP 세션을 이용한다


### TCP/IP 프로토콜에 대한 설명으로 옳지 않은 것은 ?


: LAN 환경에서만 사용할 수 있는 제한된 프로토콜이다


### DHCP 에 관한 설명으로 옳지 않은 것은 ?


: IP Address 가 없는 경우에도 인터넷에 접속이 가능하게 해준다


### Subnetting 을 하는 이유로 옳지 않은 것은 ?


: Host ID 를 사용하지 않아도 된다


### 어떤 전송 시스템의 전송신호의 주파수는 최소 2KHz 에서 최대 10KHz 일 때 이 신호를 손실 없이 디지털화 하기 위한 표본 추출 수는 ?


: 16,000


### 다음 중 데이터 흐름제어(Flow Control)와 전혀 무관한 것은 ?


: Loop/Echo


### 데이터통신망에서 정보를 교환하는 방식으로 옳지 않은 것은 ?


: 망 교환(Network Switching)


### 다음 중 TCP/IP 와 OSI 7 Layer 의 비교로서 옳지 않은 것은 ?


: HTTP 프로토콜은 OSI 7 Layer 의 표현계층에 해당한다


### Ethernet 에 대한 설명 중 올바른 것은 ?


: CSMA/CD 프로토콜을 이용한다


### 프로토콜 계층구조상의 기본 구성요소로 옳지 않은 것은 ?


: 구문 (Syntax)


### 다음 중 일반적으로 Finger 를 통해 알 수 없는 정보는 ?


: 해당 사용자의 비밀번호


### 다음 중 고속 Ethernet 의 엑세스 방식에 해당되는 것은 ?


: CSMA/CD


### W 에서 프린터를 공유하여 같이 사용할 때 프린트를 요청했지만 프린트가 되지 않고 작업을 지울 수도 없을 때 문제를 해결하기 위해 이용하는 서비스는 ?


: Spooler Service


### W 에서 웹 서비스 및 FTP 서비스를 하려고 할 때 먼저 설치되어야 하는 것은 ?


: IIS Web Server


### DNS 서버를 처음 설치하면서 가장 먼저 만들어야 하는 데이터베이스 레코드는 ?


: SOA(Start Of Authority)


### 익명의 사용자가 회사 FTP 사이트의 특정 디렉터리에 파일을 업로드 할 수 있도록 하기 위해서 적용할 사용 권한은 ?


: 특정 디렉터리에 쓰기 사용권한을 부여한다


### SQL Server 를 관리하는데 있어서 편리한 사용자 인터페이스를 제공하는 툴로 트랜잭트(Transacted) SQL 문을 모르더라도 데이터베이스나 테이블을 생성하는 작업을 손쉽게 할 수 있으며 데이터 베이스 백업 및 복구 작업을 할 수 있는 것은 ?


: 엔터프라이즈 매니저


### DHCP 관리자는 특정 IP Address 를 특정 호스트가 예약하여 사용하는 것을 허용한다 예약 설정을 하기 위해서 필요한 항목은 ?


: 해당 호스트의 MAC Address


### DHCP 클라이언트가 DHCP 서버로부터 IP 를 임대 받는 과정 중 처음으로 초기화할 때 클라이언트가 서버에 대하여 패킷을 브로드캐스트 하는 단계는 ?


: IP 대여 요청단계


### W 에서 Active Directory 는 모든 네트워크 자원들에 대한 중앙 집중적인 관리를 지원한다 그러므로 사용자는 단 한 번의 로그온으로 네트워크상의 어떠한 자원들이라도 엑세스 할 수 있다 이때 Active Directory 설치 마법사 실행 파일은 ?


: dcpromo.exe


### W 설치 시에 기본적으로 설정되는 그룹이 아닌 것은 ?


: Operators


### NTFS 에 대한 설명으로 옳지 않은 것은 ?


: NTFS 는 W 시스템에서 가장 효율적인 파일 시스템으로 대부분의 운영체제에서 호환성을 가진다


### FAT 와 NTFS 에 대한 설명 중 옳지 않은 것은 ?


: FAT 는 자체적인 압축 기능을 지원한다


### W 의 인증 프로토콜끼리 짝지어진 것은 ?


: EAP, MS-CHAP


### 조직 내에 극도의 보안을 요구하는 팀이 한시적으로 결성되어 이들에 대해서는 패스워드를 일주일에 한 번씩 변경하게 하려고 한다 어떤 단위로 그들을 묶는것이 가장 적절한가 ?


: OU(Organization Unit)


### IIS 를 설치할 경우 가능한 서비스가 아닌 것은 ?


: DHCP 서비스


### Linux 시스템에 설치된 입출력 장치들과 마운트 될 파일시스템의 마운트 포인터가 위치하는 디렉터리는 ?


: /mnt


### 다음 중 Linux 에서 DNS 서버의 설정과 관련된 파일은 ?


: /etc/named.conf


### Linux 를 설치하고 부팅하였더니 li 메시지만 나오고 시스템이 멈추었다 이 때 lilo 를 삭제하는 방법은 ?


: fdisk/mbr 명령어를 사용하여 부트 레코드를 제거한다


### 다음 중 네트워크 어댑터 카드 (NIC) 에 대한 설명으로 옳지 않은 것은 ?


: OSI 참조 모델 중 4계층 장비이다


### 라우터(Router) 에 대한 설명으로 올바른 것은 ?


: 패킷 전송 시 논리적인 주소인 IP Address 를 기초로 하고 OSI 참조모델 중 3계층 장비이다


### 다음 네트워크 장치 중 서로 다른 프로토콜을 가진 두 시스템 간을 연결시키며 OSI 참조모델의 모든 계층에서 동작하는 장치는 ?


: Gateway


### T-1 회선에 사용되는 프레임은 보통 각각 8bit 로 된 24 Channel 과 여분의 1비트로 구성된다 이러한 T-1 회선이 초당 나눌 수 있는 프레임의 개수는 ?


: 8000


### 네트워크 회선 기기 중 브릿지에 대한 설명으로 옳은 것은 ?


: 데이터링크 계층에서 작동한다
