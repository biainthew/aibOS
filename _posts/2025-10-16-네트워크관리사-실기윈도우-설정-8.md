icqa = 서비스 관리자(Windows Sever) 선택 후 실행
원격 사용자가 Telnet을 이용하여 파일을 삭제하여 왔으나 정책이 변경되어 원격 사용자가 더 이상 로그온 할 필요가 없어졌다.
해당 기능을 중지시키고 다시 시작할 수 없게 설정하시오
서비스 페이지에 Telnet이 삭제된 것으로 추정. (서비스를 잘 찾는것이 중요함!!!)
하지만 문제를 잘 읽고 해당 기능 페이지에서 문제의 변경 설정을 잘 적용해야함!
1) 서비스 사용 유무 = 시작 유형 : 사용 안함
2) 해당 기능 사용 중지 시키기 = 서비스 상태 : 중지
