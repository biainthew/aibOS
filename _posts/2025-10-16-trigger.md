### `$(document).trigger(eventName, [param1, param2, …])`


: trigger() 메서드는 이벤트를 발생시키면서 추가적인 데이터를 핸들러로 전달할 수 있음


: 하지만 두 번째 인자 자체가 하나의 배열로 처리되므로 전달 방식에 주의해야 함


### 잘못된 예시


```javascript
$(document).trigger('received', [combinedEl], prevData.id);
```


trigger() 의 두 번째 인자는 배열 하나로만 전달되기 때문에 [prevData.id](http://prevdata.id/) 는 전달되지 않고 무시됨


### 올바른 예시


```javascript
$(document).trigger('received', [combinedEl, prevData.id]);
```


### 핸들러에서 받는 법


```javascript
$(document).on('received', function (event, combinedEl, id) {
    console.log(combinedEl); // 첫 번째 데이터
    console.log(id);         // 두 번째 데이터 (prevData.id)
});
```
