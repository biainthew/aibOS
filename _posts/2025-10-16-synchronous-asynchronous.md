# _**Synchronous 동기식**_


: 현재 작업의 응답이 끝남과 동시에 다음 작업이 요청됨


: 함수를 호출하는 곳에서 호출되는 함수가 결과를 반환할 때까지 기다림


: 작업 완료 여부를 계속해서 확인함


# _As__**ynchronous 동기식**_


: 현재 작업의 응답이 끝나지 않은 상태에서 다음 작업이 요청됨


: 함수를 호출하는 곳에서 결과를 기다리지 않고, 다른 함수(callback)에서 결과를 기다림


: 작업 완료 여부를 확인하지 않음
