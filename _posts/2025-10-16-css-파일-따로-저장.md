{% raw %}
```javascript
"liveSassCompile.settings.formats": [
    {
      "format": "expanded",
      "extensionName": ".css",
      "savePath": "~/../css",
      "savePathReplacementPairs": null
    }
  ]
```
{% endraw %}


settings.json 에 위의 코드 추가해주면 컴파일 했을 때 css 폴더가 따로 생성되고 그 안에 자동으로 저장됨
