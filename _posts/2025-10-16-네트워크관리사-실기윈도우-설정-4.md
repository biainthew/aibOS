1. 인터넷 정보 서비스 관리자 선택
2. 사이트 마우스 우클릭 -> FTP 사이트 추가
3. FTP 사이트 이름 : ICQA, 실제경로 C:\inetpub\ftproot 입력 -> 다음
4. IP주소 : 192.168.100.10, 포트:2121 입력 (선택할 수 있게 되어있어 헷갈릴 수 있는데 그냥 입력하면 됨!) -> 다음
5. 엑세스 허용 : 익명 사용자, 사용 권한 읽기 체크 (인증은 건들지 마세요 헷갈리지 마자) -> 마침
6. 사이트 하단에 생성한 ICQA 선택 -> FTP IPv4 주소 및 도메인 제한 더블클릭 -> 해당 공란에서 마우스 우클릭 -> 거부항목추가 (허용항목추가도 여기서 할 수 있음)
-> 거부 제한 규칙 추가 란에서 IP 주소 범위에 200.115.100.0, 마스크란에 255.255.255.0 (24를 10진수로!) 입력 -> 확인 -> 해당 창 나갈 때 저장안내창 뜨면 확인 누르기
7. FTP 메시지 더블클릭 -> 최대 연결 수 메시지란에 최대 접속 인원수를 초과하였습니다. 입력 -> 다른 페이지 클릭하면 저장 창 나옴!! 꼭 저장하고 나가기!!!! 매우중요!!
