| etc      | 시스템 설정 파일                                                                         |
| -------- | --------------------------------------------------------------------------------- |
| bin      | 기본적인 명령어 (Shell)                                                                  |
| home     | 사용자 계정을 만들면 home 하위에 해당 계정의 홈 디렉토리가 생성
ex) 윈도우의 내 문서, 내 사진, 내 음악 등이 있는 공간         |
| root     | root 사용자, 관리자만 사용 가능한 공간                                                          |
| tmp      | 임시저장 및 입출력파일 저장                                                                   |
| var      | 이메일, 각종 로그                                                                        |
| ls       | 파일과 디렉토리를 확인 (List 의 줄임말 정도)
비슷한 명령어 : dir                                        |
| cd       | (change directory) 디렉토리 간 이동                                                      |
| mv       | (move) 파일 이동                                                                      |
| pwd      | (print working directory) 현재 내가 있는 위치 확인                                          |
| mkdir    | (make directory) 디렉토리 생성                                                          |
| rmdir    | (remove directory) 디렉토리 삭제                                                        |
| rm       | (remove) 파일 삭제                                                                    |
| cp       | (copy) 파일이나 폴더 복사
ex ) cp[옵션][원본데이터][대상파일이나 디렉토리명]                                |
| clear    | 현재 보이는 화면 지울 때                                                                    |
| man      | (manual) 명령어는 알고 있지만 사용 방법을 모르는 경우
명령어 앞에 붙임                                      |
| su       | 사용자 변경 (다른 ID 로 변경)
su 는 관리자 root 로 전환 sudo 는 관리자 권한을 빌림                          |
| cat      | 간단한 파일의 내용을 살펴보거나 다른 파일 내용을 결합시킬 때 사용
비슷한 명령어 : vi(view)
cat 은 내용을 보기만 vi 는 편집 가능 |
| vi       | 유닉스 시스템에서 주로 사용하는 텍스트 편집기                                                         |
| head     | 파일의 앞부분을 기준으로 보기                                                                  |
| tail     | 파일의 뒷부분을 기준으로 보기                                                                  |
| find     | 특정 파일을 찾을 때
find[경로][-옵션][찾는 파일명][action] 형식으로 사용                                 |
| grep     | (global regular expression print) 텍스트 파일의 내용에서 원하는 단어나 문장이 포함된 행을 찾아주는 명령어        |
| shutdown | 서버 종료                                                                             |
| dd       | 경로 이동                                                                             |
| chage    | 계정 패스워드 관리 (change 아님)                                                            |
| free     | 관리중인 시스템의 메모리 상태 체크                                                               |
| ipconfig | 네트워크 인터페이스의 설정 내용 확인                                                              |
| wc       | 단어 수, 줄 수를 카운트 (word count)                                                       |
| tar      | (tape archive) 파일을 압축하거나 풀 때                                                      |
| groupadd | 그룹 추가                                                                             |
| groupmod | 그룹 수정                                                                             |
| groupdel | 그룹 삭제                                                                             |
| top      | 실시간으로 프로세스를 모니터링하는 화면 출력                                                          |
| df       | (disk free) 디스크의 남은 용량 확인 (각종 정보 확인, 마운트 위치 등)                                    |
| du       | (disk usage) 디스크 사용량                                                              |
| finger   | 사용자 정보                                                                            |
| history  | 이전 사용 명령어                                                                         |
| cal      | 달력 출력                                                                             |
| hostname | 현재 사용중인 시스템 이름 표시                                                                 |
| install  | 파일 설치                                                                             |
| last     | 마지막 사용자의 접속 상태(정보) 확인                                                             |
| leave    | 자신의 접속 종료시간 표시                                                                    |
| lp       | 인쇄 시작                                                                             |
| lpq      | 인쇄작업 상황보기                                                                         |
| lpm      | 인쇄작업 지우기                                                                          |
| users    | 현재 접속 사용자 보기                                                                      |
| who      | 시스템에 접속되어 있는 사람만 보여줌                                                              |
| rwho     | 현재 접속자 보기                                                                         |
| rwall    | 모든 사용자에게 알림                                                                       |
| size     | 오브젝트 파일의 크기 확인                                                                    |
| whatis   | 명령어의 간단한 설명                                                                       |
| whereis  | 찾는 명령어의 실행파일, 소스, man 페이지 안내                                                      |
| time     | 현재 시간                                                                             |
| nslookup | name server 와 관련된 조회 가능                                                           |
| usermod  | 사용자 계정 수정                                                                         |
| fdisk    | 설치된 하드디스크의 타입과 정보 확인 (편집 가능)                                                      |
| file     | 파일의 종류 (type) 를 확인할 때                                                             |
| chmod    | 파일의 허가권을 변경                                                                       |
| chown    | 파일의 소유권을 변경                                                                       |
| chgrp    | 파일의 그룹 소유권을 변경                                                                    |
| crontab  | 작업설정 파일                                                                           |
| ps       | 데몬(프로세스)이 살아 있는지의 여부를 확인                                                          |
| logout   | 로그인 상태에서 시스템을 종료하는 것이 아니라 자신만 리눅스에서 빠져나옴                                          |
| kill     | 현재 동작중인 프로세스 종료                                                                   |
| ping     | 해당 사이트와의 통신 상태를 점검할 때 사용                                                          |
| pipe     | 한 프로그램의 출력을 중간 파일 없이 다른 파일의 입력으로 바로 보내는 유닉스 매커니즘                                  |
| netstat  | 네트워크 연결상태, 라우팅 테이블, 인터페이스 관련 통계 정보 등을 확인
현재 열려 있는 포트도 확인 가능                       |
| umask    | 파일 및 디렉토리 자동 권한 설정                                                                |
