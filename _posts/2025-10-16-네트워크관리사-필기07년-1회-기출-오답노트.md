### 이더넷에서 MAC 프레임의 발신지 주소는 ?


: 최초 송신자의 물리 주소


### TCP/IP 에 대한 설명으로 옳지 않은 것은 ?


: IP 는 데이터의 에러 검출을 담당한다


### Transport 계층에서 TCP와 UDP 프로토콜의 차이점으로 옳지 않은 것은 ?


: UDP 는 TCP 에 비해 오버헤드가 크다


### 도메인 네임으로 IP Address 를 조회할 수 있는 명령어는 ?


: Nslookup


### 인터넷에서 사용되는 네트워크 B Class 에서 한 네트워크 내의 최대 호스트 수는 ?


: 65,534


### IP Address 에 대한 설명으로 옳지 않은 것은 ?


: B Class 의 최상위 2비트는 “01” 로 설정한다


### IPv6에 대한 설명으로 옳지 않은 것은 ?


: 현재의 IPv4와 상호 운용이 불가능하다


### Subnetting 을 하는 이유로 옳지 않은 것은 ?


: Host ID 를 사용하지 않아도 된다


### 정적 라우팅의 사용이 가장 적절한 환경은 ?


: 네트워크 규모가 작고 다른 네트워크에 대한 접속점이 하나이고 경로가 이중화되있지 않은 조건


### 연결 없는 IP 기반의 프로토콜로 최소한의 오버헤드를 갖는다, 재송신 처리를 실행하지 못하기 때문에 신뢰성이 떨어진다, 한번에 많은 양의 데이터를 송신할 때 사용한다


: UDP


### 동적 주소 바인딩을 위한 인터넷 표준 IP Address 로부터 MAC Address 를 얻기 위한 프로토콜은 ?


: ARP


### FTP 가 사용하는 두 개의 TCP 포트를 바르게 짝지은 것은 ?


: 20 - 데이터 전송, 21 - 제어용


### 라우터를 이용해서 Segment-1 과 Segment-2 로 네트워크를 나누었을 때 나타나는 결과로 올바른 것은 ?


: Segment-1 의 브로드캐스트가 Segment-2 로 들어가는 것을 막아준다


### 다음 중 장비와 OSI 7 Layer 가 가장 올바르게 연결된 것은 ?


: 스위치 - DataLink Layer


### 네트워크에 관련한 표준 제정 기관으로 옳지 않은 것은 ?


: OSIA


### DCE(Data Communications Equipment) 범주에 속하지 않는 것은 ?


: TDM (Time Division Multiplexing)


### 4위상 변조를 갖는 변조기가 2400Baud 의 변이가 일어날 때 속도는 ?


: 4800bps


### Multiplexing 방법 중에서 다중화시 전송할 데이터가 없더라도 타임 슬롯이 할당되어 대역폭의 낭비를 가져오는 다중화 방식은 ?


: TDM


### 다음 중 CATV 의 주요 구성 요소로 가장 옳지 않은 것은 ?


: 엑세스 포인트


### 패킷 교환망의 특징이 아닌 것은 ?


: 망에 유입되는 데이터의 양이 많아질 수록 전송속도가 빠르다


### LLC 계층과 MAC 계층으로 구분되는 계층은 ?


: 데이터 링크 계층


### 웹 브라우저에서 인터넷과 인트라넷을 경유한 프린팅을 가능하게 해주는 프로토콜은 ?


: IPP


### 일반 사용자들이 Print Server 에 프린트 작업을 요청했지만 프린팅이 되지 않고 작업을 지울 수도 없는 경우 이러한 문제를 해결하기 위하여 이용할 수 있는 서비스는 ?


: Spooler Service


### 호스트 네임과 도메인 네임이 함께 붙여진 이름을 뜻하는 것은 ?


: FQDN


### DNS 서버에서 해당 도메인의 하위메뉴에 대한 설명이다 옳지 않은 것은 ?


: 메일 교환기 탭의 메일 서버 우선순위는 높은 번호가 우선적으로 제공된다


### W 의 DNS 서버는 기본적으로 레지스트리에 저장된 정보를 사용하여 서비스를 초기화하고 서버에서 사용할 영역 데이터를 로드한다 추가된 옵션으로 파일에서 부팅하도록 구성하는 경우 사용할 파일은 ?


: BOOT 텍스트 파일


### W 에 [ftp.icqa.or.kr](http://ftp.icqa.or.kr/) 의 FTP 가 운영되는 동일 컴퓨터에 [ftp.network.or.kr](http://ftp.network.or.kr/) 의 FTP 사이트를 운영하려고 할 때 다음 중 가장 적절하지 못한 조치는 ?


: 별도의 호스트 헤더를 설정한다


### W 의 FTP 사이트를 처음 개설하면 멈춤 상태에 있게 된다 사용자들이 FTP 접속을 할 수 있도록 하려면 어떤 버튼을 클릭해야 하는가 ?


: 시작


### RDP(Remote Desktop Protocol) 에 대한 설명으로 옳지 않은 것은 ?


: RDP 를 이용한 터미널 서비스는 LAN 상에서만 연결을 지원한다


### W 에서 사용하는 관리자 프로그램에 대한 인터페이스를 표준화시킨 것으로 서버에서 사용하는 모든 관리자 도구로 서버 구성요소 각각에 대해서 스냅인을 해서 항목을 추가하거나 삭제하는 것은 ?


: MMC(MicroSoft Management Console)


### 터미널 서비스를 사용하기 위한 정적 포트가 잘못 짝지어진 것은 ?


: RDP 클라이언트 (Microsoft) TCP : 4662


### 다음 중 OU(Organization Unit) 에 대한 설명으로 올바른 것은 ?


: Active Directory 에서 각종 보안 및 정책 관리의 최소 단위


### 그룹 계정을 사용하는 이유는 그룹에 권한을 설정하여 사용자 계정을 포함시키면 관리하기 편리해지기 때문인데 그룹 계정의 종류에 해당하지 않는 것은 ?


: Network Group


### W 에서 스팬 볼륨 에 대한 설명으로 옳지 않은 것은 ?


: 미러 볼륨을 만들 수 있다


### 물리적 네트워크의 인터페이스를 시험하고 로컬호스트에서 NIC 의 올바른 동작여부를 검사할 수 있는 명령은 ?


: Ipconfig


### UDP 에 대한 설명으로 가장 옳지 않은 것은 ?


: 양방향 전송을 하며 종단간의 흐름 제어를 위해 Dynamic Sliding Window 방식을 사용한다


### IP(Internet Protocol) 를 직접 이용하는 응용 프로그램은 ?


: Traceroute


### ARP Packet Frame 과 RARP Packet Frame 은 거의 같은 Packet Format 을 사용한다 그러나 이 Packet Format 에서 두 가지 프로토콜이 서로를 구별하기 위하여 다르게 사용하는 필드가 있는데 이들로 이루어진 것은 ?


: Frame Type, Op


### TCP/IP 프로토콜 군(Protocol Suite) 에 대한 설명으로 옳지 않은 것은 ?


: IP 는 네트워크에서 호스트 지정과 라우팅의 안전성을 위해 연결형 (Connection Oriented) 프로토콜로 동작한다


### 라우팅 프로토콜에 대한 설명으로 옳지 않은 것은 ?


: Cisco Router 의 IGRP 에 대한 Default Administrative Distance 값은 110 이다


### 서브넷 마스크에 대한 설명으로 옳지 않은 것은 ?


: 서브넷 마스크는 Network IP 필드는 0 으로 Host ID 의 필드는 1 로 채운다


### Routed Protocol 이란 네트워크 상에서 Route 되는 프로토콜이다 Routed Protocol 로 옳지 않은 것은 ?


: RIP


### 라우터에서 패킷의 목적지를 결정하는 방법은 ?


: 해당 서브넷 마스크와 AND 연산이 이루어진다


### IP 헤더 구조 중 인터넷에 생존할 수 있는 최대시간을 나타내는 필드로 이 값이 0 이 되는 데이터그램은 파기된다


: TTL


### RARP 에 대한 설명으로 가장 옳지 않은 것은 ?


: ARP 와 같은 역할을 수행하며 로컬 디스크가 없는 시스템에 주로 사용된다


### 인터넷 그룹 관리 프로토콜로 컴퓨터가 멀티캐스트 그룹을 인근의 라우터들에게 일리는 수단을 제공하는 인터넷 프로토콜은 ?


: IGMP


### 네트워크를 공간적인 거리에 따라 구분하는 방법 중 옳지 않은 것은 ?


: VAN


### 패킷교환의 특징에 대한 설명 중 가장 옳지 않은 것은 ?


: 복수의 상대방과는 통신이 불가능하다


### Routing 정책에 대한 다음 설명 중 옳지 않은 것은 ?


: Random Routing 은 네트워크 정보를 이용하지 않기 때문에 트래픽 부하를 높일 수 있다


### RS-232 통신의 설명으로 옳지 않은 것은 ?


: 경제적이며 원거리 통신에 적합하다


### URL 은 어떤 용어의 약어인가 ?


: Uniform Resource Locator


### 공중 통신 사업자로부터 통신 회선을 임대하여 컴퓨터를 접속시킨 후 이 컴퓨터가 갖고 있는 정보를 여러 사용자에게 재판매하는 통신 서비스 시스템은 ?


: VAN (Value Added Network)


### W 의 특징으로 올바른 것은 ?


: 마스터 파일 테이블이 존재하여 이에 대한 복제와 파일 로그가 유지되므로 파일 복구가 가능하다


### DNS 서버와 가장 동일한 역할을 수행하는 것은 ?


: LMHOSTS


### W DNS 서버 중 주기적으로 보조 서버에 정보를 제공하는 서버는 ?


: 주 서버


### W 에서 FTP 서버의 등록정보에 없는 탭은 ?


: 성능


### FTP 서버에 대한 설명으로 옳지 않은 것은 ?


: W 의 기본 Port 는 23번을 사용한다


### W 의 서비스 중 Windows NT Server 에는 없는 새로운 서비스는 ?


: Active Directory


### W 의 DHCP 구성 시 같은 IP Address 를 사용하기 위해 IP Address 를 예약한다 이때 설정되는 사항으로 옳지 않은 것은 ?


: 서브넷마스크


### 다음 중 DACL(Directory Access Control List) 에 대한 설명으로 가장 올바른 것은 ?


: 한 개체에 대한 접근 제어 항목의 리스트로 위임 할 때 사용된다


### W 에서 원격 설치 서비스를 구성하기 위하여 원격설치서버 상에 구현이 되어 있어야 하는 네트워크 서비스로 가장 옳지 않은 것은 ?


: SMTP


### 한 개의 하드디스트에 W 와 Windows 98 로 멀티부팅을 사용하기 위한 시스템 설정으로 가장 올바른 것은 ?


: 하드디스트의 주 파티션을 FAT 로 포맷한다


### W 에서 리소스 레코드로 옳지 않은 것은 ?


: DN


### 프로토콜 중 일반적으로 UDP 를 사용하는 것으로 가장 올바른 것은 ?


: TFTP


### Linux 시스템 커널 버전에 대한 규칙으로 옳지 않은 것은 ?


: 버전의 두 번째 자리가 홀수이면 안정 버전 짝수이면 개발 버전을 뜻한다


### Linux 에서 사용되는 용어 중 화면으로 출력될 내용을 파일로 저장하거나 프린트하고자 할 때 사용하는 기호는 ?


: >


### RAID 에 대한 특징 중 옳지 않은 것은 ?


: 운영체제에서 여러 개의 논리적 드라이브를 하나의 물리적 드라이브로 활용한다


### Cisco 2501 Router 의 설정을 바꾸기 위해 네트워크 관리자가 Router 에 접속하기 위해 사용할 수 있는 방법으로 옳지 않은 것은 ?


: In-Line 포트를 이용하여 연결한다


### 스위치 허브의 전이중 방식과 반이중 방식에 대한 설명 중 올바른 것은 ?


: 100Mbps 를 갖는 전이중 방식의 스위치 허브 대역폭은 200Mbps 이다


### 어떤 회사에서 라우터를 이용해서 네트워크를 구축하려고 한다 내부 네트워크는 모두 같은 네트워크로 만들고 외부로는 전용선을 통해 네트워크를 구축한다고 했을 때 필요한 라우터의 인터페이스로 가장 올바른 것은 ?


: 한 개의 이더넷 인터페이스와 한 개의 시리얼 인터페이스


### LAN Card 의 IP Address 를 세팅할 때 디폴트 게이트웨이 주소는 일반적으로 어디 주소를 의미하는가 ?


: 해당 PC 가 접속되어 있는 라우터의 IP Address
