### W 에서 동적 호스트 구성 (DHCP), DNS 등의 서버 주소를 포함하여 서브넷 마스크, 기본 게이트웨이 등 TCP/IP 구성 파라미터를 확인할 수 있는 명령어는 ?


: Ipconfig


### 다음 중 Transport 계층에서 TCP 와 UDP 프로토콜의 차이점을 잘못 설명한 것은 ?


: UDP 는 TCP 에 비해 오버헤드가 크다


### TCP/IP Protocol 군에서 네트워크 계층의 프로토콜로만 연결된 것은 ?


: ICMP-IP-IGMP


### IP Address 127.0.0.1 이 의미하는 것은 ?


: 루프백 테스트 용이다


### 서브넷의 기능에 대한 설명으로 옳지 않은 것은 ?


: 네트워크 ID 를 두 부분으로 나누어 각 세그먼트에 대해 고유한 서브넷 ID 를 생성한다


### TCP 와 UDP 에 대한 설명으로 옳지 않은 것은 ?


: TCP 와 UDP 모두 데이터 수신에 대한 긍정응답 (ACK) 을 사용한다


### 프로토콜의 부하가 적어 분산처리에서 많이 사용되는 인터넷 프로토콜은 ?


: UDP


### ICMP 프로토콜에 대한 설명으로 올바른 것은 ?


: IP 데이터그램을 사용하지만 메시지는 TCP/IP 소프트웨어에 의해 처리된다


### WWW 에 대한 설명으로 옳지 않은 것은 ?


: 23번 포트를 사용한다


### 인터넷의 잘 알려진 포트번호로 옳지 않은 것은 ?


: TFTP - 68


### FTP 와 Telnet 의 공통점을 열거한 것으로 옳지 않은 것은 ?


: 필요에 따라 TCP 나 UDP 로 선택할 수 있다


### TCP/IP 계층 중 다른 계층에서 동작하는 계층은 ?


: SMTP


### 메시지 교환 방식의 특징으로 옳지 않은 것은 ?


: 데이터 전송 지연 시간이 거의 없다


### OSI 7 Layer 중 서로 다른 네트워크의 차이를 극복하여 종점 간에 신뢰성 있고 투명한 데이터 전송을 제공하고 종점간의 에러 복구와 흐름 제어를 하는 것은 ?


: 트랜스포트 계층


### IEEE802.3 표준안을 포함하고 있는 것은 ?


: Ethernet


### 네트워크 케이블에서 실제로 움직이는 패킷은 크게 3섹션으로 구별된다 다음 중 이 패킷의 구성이 아닌 것은 ?


: 데이터 그램


### 정보 교환 네트워크에서 이용자의 데이터를 일정단위로 나누어 각각에 송수신 주소를 부가하여 전송하는 방식은 ?


: 패킷 교환방식


### 모든 노드를 중앙에서 관리하며 하나 이상의 노드에 이상이 발생하여도 전체 네트워크에 문제가 발생하지 않는 형태의 토폴로지는 ?


: 스타 토폴로지


### 전송 채널 엑세스를 하기 위한 방식들이다 각 노드에서 채널의 상태를 감지하여 데이터의 충돌을 방지할 수 있는 엑세스 방식은 ?


: CSMA


### VolP 구성요소 중에서 Client 에 속하지 않는 것은 ?


: Gate Keeper


### W 가 지원하는 File System 으로 옳지 않은 것은 ?


: EXT2


### W 의 인터넷 정보 서비스(IIS) 에서 사용 가능한 서비스는 ?


: FTP 서비스, 웹 서비스


### 명령 프롬프트에서 IIS 서비스를 시작하기 위한 명령은 ?


: net start w3svc


### NetBIOS 이름을 IP Address 로 매핑하기 위해 사용되는 서비스는 ?


: WINS


### 호스트 네임과 도메인 네임이 함께 붙여진 이름을 뜻하는 것은 ?


: FQDN fully qualified domain name


### 새로운 FTP 사이트를 생성하기 위한 FTP 사이트 만들기 마법사 설명 중 가장 옳지 않은 것은 ?


: 설명(D) 의 입력란에는 반드시 호스트명을 사용한다


### W 의 FTP 사이트를 처음 개설하면 멈춤 상태에 있게 된다 사용자들이 FTP 접속을 할 수 있도록 하려면 어떤 버튼을 클릭해야 하는가 ?


: 시작


### TCP/IP 주소 관리를 간소화 하기 위하여 자동 IP 를 할당하고자 할 때 사용하는 서비스는 ?


: DHCP dynamic host configuration protocol


### Active Directory 의 논리적 구조 중 도메인이 제공하는 장점으로 옳지 않은 것은 ?


: 도메인 또는 조직 단위에 관리 권한을 위임하면 모든 관리 권한을 가진 관리자가 많이 필요하다


### W 의 새 사용자 만들기 옵션에 대한 설명으로 옳지 않은 것은 ?


: 암호 사용 기간 제한 없음 - 주로 사용자 계정에서 사용하며 해당 사용자만이 암호를 변경할 수 있다


### 계정 정책은 사용자 계정에만 적용된다 계정 정책의 보안 영역에 속하지 않는 속성은 ?


: 운영 정책


### W 에서 그룹의 권한에 대한 설명 중 옳지 않은 것은 ?


: Replicator 그룹 멤버들에게 관리 업무를 할 수 있는 제한적 권한을 부여


### W 의 서비스 중 UDP 를 사용하는 것으로 가장 올바른 것은 ?


: DNS


### W 가 기본적으로 지원하지 않는 백업 장치는 ?


: 16mm DAT(Digital Audio Tape) 드라이브


### W 에서 원격 설치 서비스를 구성하기 위하여 원격설치서버 상에 구현이 되어 있어야 하는 네트워크 서비스로 옳지 않은 것은 ?


: SMTP


### 다음 중 Linux 시스템에서 가장 널리 사용되고 있는 파일 시스템은 ?


: EXT2


### Linux 시스템에서 데몬에 관한 설명 중 옳지 않은 것은 ?


: 시스템 부팅 때만 시작될 수 있다


### Linux 시스템에서 현재 디렉터리와 하부 디렉터리까지 검색하여 core 라는 파일을 찾아 삭제하는 명령은 ?


: find . -name core -exec rm -rf {} ;


### T3 회선에서의 데이터 전송속도는 ?


: 45Mbps


### 현행 전화선으로 높은 대역폭으로 디지털 정보를 전송하며 상하향 데이터의 전송 속도가 다른 기술은 ?


: ADSL asymmetric digital subscriber line


### 최근 공중망 무선 랜 서비스가 상용화 되었는데 가입자는 접속장치에서 100m 반경 이내에서 무선 랜 카드 장착 노트북 PDA 등을 통해서 서비스를 받을 수 있는 접속장치는 ?


: AP(Access Point)
