### 접근 제어자(Access Modifiers)


접근 제어자는 클래스, 메서드, 변수에 대해 외부에서 접근할 수 있는 범위를 지정한다


자바에는 네 가지 접근 제어자가 있으며 각각의 접근 수준은 다음과 같다


| 접근 제어자      | 클래스 내 | 같은 패키지 | 하위 클래스 | 전체(외부) |
| ----------- | ----- | ------ | ------ | ------ |
| `public`    | ☑️    | ☑️     | ☑️     | ☑️     |
| `protected` | ☑️    | ☑️     | ☑️     | ❎      |
| `default`   | ☑️    | ☑️     | ❎      | ❎      |
| `private`   | ☑️    | ❎      | ❎      | ❎      |


**특징**

1. public
    1. 어디서든 접근할 수 있다 다른 패키지나 클래스에서도 접근 가능하며 프로젝트 전체에서 사용할 수 있다
    2. 주로 라이브러리 API 처럼 외부에서도 자주 사용해야 하는 메서드와 클래스에 사용한다
2. protected
    1. 같은 패키지 내에서 자유롭게 접근할 수 있으며 다른 패키지의 하위 클래스에서도 접근이 가능하다
    2. 주로 상속받은 클래스에서만 접근하도록 제한하면서도 패키지 내에서의 접근이 필요할 때 사용된다
3. default
    1. 접근 제어자를 명시하지 않으면 기본 접근 수준이 적용된다 같은 패키지 내에서만 접근 가능하며 패키지 외부에서는 접근할 수 없다
4. private
    1. 해당 클래스 내에서만 접근할 수 있다 다른 클래스에서는 접근이 불가능하다
    2. 변수, 메서드가 클래스 내부에서만 사용될 경우 외부에 공개하지 않고 보안을 유지하기 위해 사용한다
