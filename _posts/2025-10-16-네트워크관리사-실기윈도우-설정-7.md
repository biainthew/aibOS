icqa = 로컬 보안 정책 선택 후 실행
보통 계정 정책 안에서 암호 정책과 계정 잠금 정책에서 다 끝난다 생각하면 됨

1. 패스워드는 최소 10일에서 최대 20일 사용 = 암호 정책 페이지에서 확인
    1. 최소 암호 사용기간 더블클릭 -> 10일 입력 후 확인
    2. 최대 암호 사용기간 더블클릭 -> 20일 입력 후 확인
2. 패스워드는 3번 로그인 실패시 30분간 계정 잠금 = 계정 잠금 정책 페이지에서 확인
    1. 계정 잠금 임계값 -> 3번의 잘못된 로그인 시도로 변경 후 확인 -> 관련 변경된 내역창 뜨면 확인
    2. 계정 잠금 기간 -> 30분간 설정(기본 default 설정) 확인
