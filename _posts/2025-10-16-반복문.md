### 1. 향상된 for문


**장점**


기존의 for문보다 간결하고 코드 가독성이 좋음


**단점**


인덱스가 제공되지 않음


**형식**


```java
for (타입 변수명 : 배열 또는 컬렉션) {
	// 실행할 코드
}
```


**예시 1**


```java
int[] numbers = {1, 2, 3, 4, 5};

for (int number : numbers) {
	System.out.println(number)
}
```


**예시 2**


```java
List<String> names = Arrays.asList("Alice", "Bob", "Charlie");

for (String name : names) {
	System.out.println(name);
}
```

### 3. while문


while 문은 반복 조건이 참일 때 계속해서 반복된다 조건이 거짓이 되면 반복이 종료된다 반복 횟수가 불명확하거나 조건에 따라 반복을 계속해야 할 때 유용하다


```java
while (조건식) {
	// 반복할 코드
}
```


**예시**


```java
int i = 0;
while (i < 5) {
	System.out.println(i);
	i++;
}
```


### 4. do-while 문


while 문과 유사하지만 조건을 반복문 뒤에 검사한다 즉 최소한 한 번은 무조건 실행된다


```java
do {
	// 반복할 코드
} while (조건식);
```


**예시**


```java
int i = 0;
do {
	System.out.println(i);
	i++;
} while (i < 5);
```


### 요약


**for문** : 반복 횟수가 명확할 때


**향상된 for 문** : 배열이나 컬렉션을 간편하게 순회할 때


**while 문** : 조건이 참일 동안 반복, 조건이 언제 변할지 모를 때


**do-while 문** : 최소 한 번은 실행이 필요할 때
