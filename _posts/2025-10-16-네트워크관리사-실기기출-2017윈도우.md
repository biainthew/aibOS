**문제6**

1. DNS 선택 - 실행
2. 정방향 조회 영역 선택 - 마우스 우클릭 - 새 영역 - 다음 - 다음
영역이름 [icqa.or.kr](http://icqa.or.kr/) (ns와 admin에 공통적으로 써있는 부분만 입력) - 다음 - 다음 - 다음 - 마침
3. 생성한 [icqa.or.kr](http://icqa.or.kr/) 폴더 선택 후 마우스 우클릭 - 속성 - SOA(권한 시작) 페이지
일련번호 = serial number = 12
주 서버 = [ns.icqa.or.kr](http://ns.icqa.or.kr/)
책임자 = [admin.icqa.or.kr](http://admin.icqa.or.kr/)
새로 고침 간격 = refresh = 900
다시 시도 간격 = retry = 600
다음 날짜 이후에 만료 = expire = 86400
최소(기본값) TTL = minimum TTL = 7200
단위는 0이 많을 경우 '초' 단위로 설정해주세요. (별도 설명이 없으면 0많을 경우 '초' 단위로 설정)
확인
4. 생성한 [icqa.or.kr](http://icqa.or.kr/) 폴더 선택 후 마우스 우클릭 - 새 호스트(A 또는 AAAA) 선택 (문제 중간에 A가 있는지 확인하기)
이름에 mail, ip 주소에 211.111.144.201 입력 - 호스트 추가
이름에 network, ip 주소에 211.111.144.200 입력 - 호스트 추가 - 완료

> 💡 **A 란 ?**  
> 도메인 주소와 서버의 IP 주소 직접 매핑, 일대일은 아닐수도
