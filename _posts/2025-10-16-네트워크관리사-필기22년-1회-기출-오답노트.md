### B Class 를 6개의 네트워크로 구분하여 사용하고 싶을 때 가장 적절한 서브넷 마스크 값은 ?


: 255.255.224.0


### TCP/IP 에서 Unicast 의 의미는 ?


: 메시지가 한 호스트에서 다른 한 호스트로 전송되는 패킷


### IP 프로토콜에 관한 설명으로 올바른 것은 ?


: 네트워크 계층에 속하는 프로토콜로 실제 패킷을 전달하는 역할을 한다


### 서브넷 마스크에 대한 설명으로 옳지 않은 것은 ?


: 서브넷 마스크의 Network ID 필드는 0으로 Host ID 필드는 1로 채운다


### IPv4 환경에서 사용되는 사설 IP address 에 대한 특징 중 올바르게 설명한 것은 ?


: 공인 IP address 의 부족 문제를 해결하기 위하여 사용된다


### 총 20개의 패킷을 캡쳐, http.cap 파일명으로 저장, 80번 Port 통신을 캡쳐


: tcpdump -c 20 -w http.cap port80


### 패킷의 목적지 주소와 라우팅 테이블의 entry 중 prefix mask 길이를 고려하여 해당 패킷을 forwarding 한다


: Longest match rule


### K 라는 회사에서 인터넷 전용회선의 대역폭을 효율적으로 제어하지 못하여 업무 마비까지 이르게 되는 현상이 발생하였다 이에 네트워크 담당자 Park 사원은 A 를 도입하여 회사의 IP 및 프로토콜이 이 장비를 반드시 통과하게 만들어서 인터넷 전용회선의 대역폭을 회사의 이벤트에 알맞도록 조정할 수 있게 되었다 평소에는 전용회선 1G 의 대역폭 중에 웹, FTP, 멀티미디어로 사용하다가 화상회의를 해야 하는 경우에는 대역폭을 조정하여 사용하고 있다 화상회의의 원활한 진행을 위하여 멀티미디어의 사용 대역폭을 증설하여 화상회의를 진행시킨 후 화상회의가 종료되는 시점에 인터넷 대역폭을 원래대로 원상복구 시킨다


: QoS (Quality of Service)


### 다음 내용이 나타내는 매체 방식은 ?
자신 외의 다른 송신자가 네트워크를 사용하는지를 점검한다, 네트워크를 아무도 사용하지 않는다면 바로 패킷을 전송한다, 패킷이 충돌하게 되면 노드는 충돌신호를 전송한 후 설정된 시간만큼 기다린 후 바로 다시 전송한다


: CSMA/CD


### 네트워크의 구성에서 성형(Star)에 관한 설명으로 옳지 않은 것은 ?


: 하나의 단말장치가 고장나면 전체 통신망에 영향을 줄 수 있다


### 클라우드 컴퓨팅 서비스들 중 하나의 유형으로서 웹 브라우저를 통하여 소프트웨어를 제공하며 서비스의 대상자는 주로 일반 소프트웨어 사용자인 것은 ?


: SaaS


### W 백업을 실행하는 방법으로 올바르지 않은 것은 ?


: 시작 - 실행 - diskpart 명령을 실행


### W 에서 Active Directory 를 구축하여 관리의 편리성을 위해 그룹을 나누어 관리하고자 한다 이 구성원은 다른 도메인의 사용자 계정이 될 수 있으나 도메인 로컬 그룹이 접근할 수 있는 자원은 자신이 소속된 도메인에 제한된다


: Domain Local Group


### W 의 DNS 서버에서 정방향 조회 영역 설정에서 SOA 레코드의 각 필드에 대한 설명으로 옳지 않은 것은 ?


: 책임자 - 해당 영역을 관리하는 사람의 전자메일 주소다 webmaster@icqa.or.kt 형식으로 기입한다


### Linux 시스템에서 상위 디렉터리에 있는 ‘abc.txt’ 파일을 홈 디렉터리로 복사하고자 한다 다음 중 알맞은 명령어는 ?


: cp ../abc.txt ~


### 리눅스의 VI 편집기를 이용하여 파일의 내용을 수정할 때 다음 내용을 만족하는 치환 명령문은 무엇인가 ?
10행부터 20행까지 내용 중 old 문자열을 new 문자열로 수정한다
각 행에 old 문자열이 여러 개가 있어도 전부 수정한다


: :10,20s/old/new/g


### W 에서 데이터 손실 없이 여러 사이트 간에 동기 복제를 제공하며 장애가 발생하기 전에 백업 데이터로 연결을 넘길 수 있도록 서버를 구성하고자 한다 이에 적절한 서비스는 ?


: 저장소 복제


### Linux 시스템 부팅과 함께 자동으로 마운트 되어야 할 항목과 옵션이 정의되어 있는 파일은 ?


: /etc/fstab


### W 에서 사용하는 Powershell 에 대한 설명으로 옳지 않은 것은 ?


: 기존 DOS 명령은 사용할 수 없다


### W 의 DNS 구성은 정방향 조회 영역을 구성하고 DNS 조회 요구의 속도 증가와 문제 해결을 돕기 위한 역방향 조회 영역을 구성하고 있다 호스트 이름에 인트라넷 서버의 IP Address 를 매핑하려 할 때 역방향 조회 영역에서 각각의 인트라넷 서버에 대해 어떤 종류의 레코드를 생성해야 하는가 ?


: Pointer(PTR)


### Linux 에서 사용자가 현재 작업 중인 디렉터리의 경로를 절대경로 방식으로 보여주는 명령어는 ?


: pwd


### W 에 해당하는 그룹 계정 중 컴퓨터 자원에 대한 인증 속성을 관리하는 권한을 갖는 그룹은 ?


: Access Control Assistance Operators


### Linux 시스템의 기본 명령어들이 포함되어 있는 디렉터리는 ?


: /bin


### W 에서 DNS 서버에 질의하여 [icqa.or.kr](http://icqa.or.kr/) 이름을 가진 호스트의 IP Address 를 보여주는 명령어로 올바른 것은 ?


: nslookup icqa.or.kr


### A 는 여러 대의 컴퓨터에 분산된 공유 폴더를 하나로 묶어서 마치 하나의 폴더인 것처럼 사용할 수 있다 A 를 사용하면 사용자들은 네트워크 사의 여러 대의 공유 폴더를 사용할 필요가 없이 한 곳으로 집중해서 사용할 수 있다


: 분산 파일 시스템


### Linux 에서 유틸리티 설명 중 옳지 않은 것은 ?


: 사용자 계정 삭제 - delete


### 하이퍼바이저를 사용하거나 게스트 운영체제도 설치하지 않고 서버 운영에 필요한 프로그램과 라이브러리만 이미지로 만들어 프로세스처럼 동작시키는 경량화된 가상화 방식이며 실행되는 이미지는 컨테이너라고 부르며 가상화 레이어가 존재하지 않고 운영체제도 존재하지 않기 때문에 파일시스템, 네트워크 속도가 상당히 빠르다


: Docker


### Repeater 에 대한 설명으로 옳지 않은 것은 ?


: 네트워크를 확장하면서 충돌 도메인을 나누어 줄 수 있는 장비가 필요한데 이럴 때 Repeater 를 사용하여 충돌 도메인을 나누어 네트워크의 성능을 향상시킨다


### IP Header 의 내용 중 TTL 의 기능을 설명한 것으로 옳지 않은 것은 ?


: IP 패킷은 네트워크 상에서 영원히 존재할 수 있다


### IP Address ‘**11101011.10001111.11111100.11001111**’ 가 속한 Class 는 ?


: D Class


### C Class 네트워크에서 6개의 서브넷이 필요하다고 할 때 가장 적당한 서브넷 마스크는 ?


: 255.255.255.224


### NAT (Network Address Translation) 에 대한 설명으로 옳지 않은 것은 ?


: NAT 를 사용할 경우 내부 사설 IP 주소는 C Class 를 사용해야만 정상적인 동작이 가능하다


### OSI 7 Layer 에 따라 프로토콜을 분류하였을 때 다음 보기들 중 같은 계층에서 동작하지 않는 것은 ?


: SMTP


### 네트워크 장비를 관리 감시하기 위한 목적으로 TCP/IP 상에 정의된 응용 계층의 프로토콜로 네트워크 관리자가 네트워크 성능을 관리하고 네트워크 문제점을 찾아 수정하는데 도움을 주는 것은 ?


: SNMP


### 호스트의 IP Address 가 201.100.5.68/28 일 때 Network ID 로 올바른 것은 ?


: 201.100.5.64


### TCP 를 사용하는 프로토콜로 옳지 않은 것은 ?


: TFTP


### IGMP (Internet Group Management Protocol) 의 특징으로 옳지 않은 것은 ?


: 데이터의 유니캐스팅에 적합한 프로토콜이다


### 클라우드 컴퓨팅의 서비스 내용에 대한 분류가 아닌 것은 ?


: Public 클라우드


### A 란 단말이 네트워크에 접근하기 전 보안 정책 준수 여부를 검사하고 IP 및 MAC Address의 인가 여부를 검사하여 네트워크 자원에 연결된 단말의 여러가지 정보를 수집하고 수집된 정보를 바탕으로 단말들을 분류하며 분류한 그룹의 보안 위협 정도에 따라 제어를 수행한다


: NAC


### Bus Topology 의 설명 중 올바른 것은 ?


: 터미네이터가 시그널의 반사를 방지하기 위해 사용된다


### 인접한 개방 시스템 사이의 확실한 데이터 전송 및 전송 에러제어 기능을 갖고 접속된 기기 사이의 통신을 관리하고 신뢰도가 낮은 전송로를 신뢰도가 높은 전송로로 바꾸는데 사용되는 계층은 ?


: 데이터 링크 계층 (Datalink Layer)


### 패킷 교환망의 특징으로 옳지 않은 것은 ?


: 망에 유입되는 데이터의 양이 많아질 수록 전송속도가 빠르다


### Multiplexing 방법 중에서 다중화 시 전송할 데이터가 없더라도 타임 슬롯이 할당되어 대역폭의 낭비를 가져오는 다중화 방식은 ?


: TDM (Time Division Multiplexer)
