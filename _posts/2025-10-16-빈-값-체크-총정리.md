```javascript
if(something) {}
```


something 의 값이 truthy(참 같은 값) 인지 확인하는 방법


|     | true                | false           |
| --- | ------------------- | --------------- |
| 문자열 | 비어있지 않은 문자열         | “” 빈 문자열        |
| 숫자  | 0이 아닌 숫자            | 0, NaN          |
| 배열  | 비어있는 배열, 비어있지 않은 배열 |                 |
| 객체  | 비어있는 객체, 비어있지 않은 객체 |                 |
|     |                     | null, undefined |


### 배열이 비어있는지 체크


배열 length로 확인


```javascript
if (something.length === 0) {}
```


### 객체가 비어있는지 체크


Object.keys 의 length로 확인


```javascript
if (Object.keys(something).length === 0) {}
```
