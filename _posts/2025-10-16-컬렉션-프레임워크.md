자바 컬렉션 프레임워크는 데이터의 저장, 수정, 검색을 위한 인터페이스와 클래스의 모음이다


주요 인터페이스와 클래스는 다음과 같다

1. List
    1. 순서가 있는 요소들의 집합으로 중복된 요소를 허용한다
    2. 주요 구현체 : ArrayList, LinkedList, Vector

        {% raw %}
```java
        List<String> names = new ArrayList<>();
        names.add("Alice");
        names.add("Bob");
        ```
{% endraw %}

2. Set
    1. 중복을 허용하지 않는 요소들의 집합
    2. 주요 구현체 : HashSet, TreeSet

        {% raw %}
```java
        Set<String> uniqueNames = new HashSet<>();
        uniqueNames.add("Alice");
        uniqueNames.add("Bob");
        ```
{% endraw %}

3. Map
    1. 키와 값 쌍으로 이루어진 데이터 집합으로 키는 중복을 허용하지 않으며 값은 중복될 수 있다
    2. 주요 구현체 : HashMap, TreeMap, LinkedHashMap

        {% raw %}
```java
        Map<String, Integer> scores = new HashMap<>();
        scores.put("Alice", 90);
        scores.put("Bob", 85);
        ```
{% endraw %}
