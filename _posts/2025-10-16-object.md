# _Object_


: 특정 기능을 수행하는 단위체


: 예시로 휴대폰의 경우 통화하기 등의 일을 시키는 것은 method / 색상 종류 등은 property


# _Window Object_


### ✔️ 종류


|   | 종류            | 설명                 |
| - | ------------- | ------------------ |
| 1 | alert         | 경고창, 확인 버튼만 나옴     |
| 2 | confirm       | 확인, 취소 버튼이 나옴      |
| 3 | prompt        | 문자를 입력할 수 있는 칸이 나옴 |
| 4 | window.open   | 입력한 페이지가 창으로 뜸     |
| 5 | window.print  | 인쇄창이 뜸             |
| 6 | setTimeout    | 한 번만 실행            |
| 7 | setInterval   | n초마다 한 번씩 실행       |
| 8 | clearInterval | 반복을 끝내기            |


### 🌈 예시 6


> 💡 `setTimeout(function(){  
>      console.log(’5초 후 실행’)  
> },5000);`


👉🏻 5초 뒤에 콘솔창에 글자 출력


### 🌈 예시 7,8


> 💡 `let i = 0;  
> let fnc = setInterval(function(){  
>       console.log(i);  
>       if(i==3){  
>            clearInterval(fnc);  
>       }  
>       i++  
> },3000);`


👉🏻 i 는 3이 될 때까지 1초마다 계속 증가, 그 값을 콘솔창에 출력
